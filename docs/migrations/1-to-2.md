# Migrating from Nimbus 1.x to 2.x

**No migration required.** There is no breaking change across this boundary.

[← Back to API Evolution](../api-evolution.md) ·
[All migration guides](./README.md)

The 2.0.0 version number predates semver discipline in this repo — it was a
milestone marker, not an API break. Every changeset in the 2.0.0 release was
`minor` or `patch` (the semver floor was `1.1.0`), and no package in the
version-locked group declared a `major` change. Upgrading from any 1.x release
to 2.x requires no code changes.

For the full reconstruction of the early version history, see
[why `1-to-2.md` is a no-op](./README.md#why-1-to-2md-is-a-no-op).

The first major with real consumer impact is 3.0.0; see
[`2-to-3.md`](./2-to-3.md).
