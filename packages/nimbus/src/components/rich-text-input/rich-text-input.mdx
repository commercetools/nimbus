## Overview

Rich text input is made of two parts, the input where users can add text, and a
toolbar. This toolbar has been arranged to suit user's needs for text editing.

Deep dive into implementation details and access the Nimbus design library.

## Variables

Get familiar with the features.

### Toolbar features

Users can change type styles, toggle bold, italics, underline, strikethrough,
code sections, superscript, subscript, bullet points and numbered lists. Undo
and redo buttons are icon buttons.

```jsx live

  const [value, setValue] = React.useState("<p><strong>Nimbus</strong> dolor sit amet, consectetur adipiscing elit. Aliquam <em>component</em> felis, vel consistency elementum. Sed design tokens ut turpis ac <u>user experience</u> maximus. Integer accessibility vitae nibh id clarity interface tincidunt.</p>");

  return (
    <RichTextInput
      value={value}
      onChange={setValue}
      placeholder="Try formatting some text..."
    />
  );
};
```

### Minimum and maximum height

These can be fully configured as needed. Toolbar should remain sticky to the top
of the input. Set the minimum height to ensure at least a few lines of content
are visible.

```jsx live
const App = () => {
  const [value, setValue] = React.useState("<p>Nimbus dolor sit amet, consectetur adipiscing elit. Aliquam component felis, vel consistency elementum. Sed design tokens ut turpis ac user experience maximus. Integer accessibility vitae nibh id clarity interface tincidunt.</p>");

  return (
    <RichTextInput
      minHeight="7200"
      maxHeight="9600"
      value={value}
      onChange={setValue}
      placeholder="Start typing..."
    />
  );
};
```
