## Overview

A toggle button is used similarly to a checkbox or a radio button, but supports
a smaller interactive component that can be used to give a binary active state
choice to users that can be used alone or in combination with other choices. A
toggle button or toggle icon button gives a user a new way of visualizing the
information in the interface. It does not typically perform an action. Please
use a button or an icon button for these uses.

## Variables

Get familiar with the features.

### Visuals

#### Outlined

Most common styling of the toggle button.

```jsx live
const App = () => (
  <ToggleButton variant="outline">
    <Icons.PushPin />
    Product categories
  </ToggleButton>
)
```

#### Ghost

Ghost styled buttons are helpful for UI simplification.

```jsx live
const App = () => (
  <ToggleButton variant="ghost">
    <Icons.PushPin />
    Product categories
  </ToggleButton>
)
```

### Color palette

#### Primary (default)

The default colorPalette for most toggle buttons.

```jsx live
const App = () => (
  <Stack gap="400" align="center" direction="row">
    <ToggleButton>
      Primary
    </ToggleButton>
    <ToggleButton isSelected>
      Selected
    </ToggleButton>
  </Stack>
);
```

#### Neutral

Use for secondary or less prominent toggles.

```jsx live
const App = () => (
  <Stack gap="400" align="center" direction="row">
    <ToggleButton colorPalette="neutral">
      Neutral
    </ToggleButton>
    <ToggleButton colorPalette="neutral" isSelected>
      Selected
    </ToggleButton>
  </Stack>
);
```
