## Overview

Tabs are a navigational component used to organize and switch between different
sections of content within the same contextual view, allowing users to access
multiple related pages or views without navigating away from the main screen.

## Tabs vs TabNav

`Tabs` and `TabNav` look identical but serve fundamentally different purposes.

| | `Tabs` | `TabNav` |
|---|---|---|
| **Use case** | Switching content panels within a page | Page-level navigation between routes |
| **HTML semantics** | `role="tablist"` + `role="tab"` | `<nav>` + `<a>` |
| **Active state** | `aria-selected="true"` | `aria-current="page"` |
| **Keyboard** | Arrow keys (roving tabindex) | Sequential Tab key |
| **Content** | Owns associated `TabPanel` | No panel — content managed by router |
| **URL** | Optional (`href` prop) | Always (`href` required) |

Both `Tabs` and `TabNav` support client-side routing — the deciding factor is whether the component owns its content panels or delegates to the router. Use `Tabs` when the content lives alongside the tab list. Use [`TabNav`](/components/navigation/tab-nav) when each item navigates to a different route and the router manages what's rendered.

### Direction for tabs

Tabs can be oriented horizontally or vertically to suit different layout
requirements.

#### Horizontal tabs

Organize and display distinct, related sections of content immediately beneath a
common navigation bar or header, serving primarily to group content contextually
while conserving vertical screen space.

```jsx live
const App = () => (
  <Stack direction="column" gap="600">
    <Box>
      <Text fontWeight="bold" mb="300">Large</Text>
      <Tabs.Root size="lg">

          <Tabs.Tab id="general">General info</Tabs.Tab>
          <Tabs.Tab id="history">History</Tabs.Tab>
          <Tabs.Tab id="subscription">Subscription</Tabs.Tab>
          <Tabs.Tab id="downloads">Downloads</Tabs.Tab>
        </Tabs.List>
        <Tabs.Panels pt="400">
          <Tabs.Panel id="general">General information content goes here.</Tabs.Panel>
          <Tabs.Panel id="history">History content goes here.</Tabs.Panel>
          <Tabs.Panel id="subscription">Subscription content goes here.</Tabs.Panel>
```
