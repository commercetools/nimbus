# Tabs

## Overview

Tabs component

```jsx-live
const App = () => (
  <Tabs.Root size="md" defaultSelectedKeys={["tab1"]}>
    <Tabs.List>
      <Tabs.Tab id="tab1">Tab 1</Tabs.Tab>
      <Tabs.Tab id="tab2">Tab 2</Tabs.Tab>
      <Tabs.Tab id="tab3">Tab 3</Tabs.Tab>
    </Tabs.List>
  </Tabs.Root>
)
```
