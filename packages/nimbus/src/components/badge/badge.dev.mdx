## Getting started

### Import

```tsx
import { Badge, type BadgeProps } from '@commercetools/nimbus';
```

### Basic usage

The simplest implementation uses the default size and color:

```jsx-live-dev
const App = () => (
  <Badge>Default Badge</Badge>
)
```

## Usage examples

### Size options

The `2xs`, `xs`, and `md` size variants are available to match your interface density:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400" alignItems="center">
    <Badge size="2xs">2xs Badge</Badge>
    <Badge size="xs">xs Badge</Badge>
    <Badge size="md">md Badge</Badge>
  </Stack>
)
```

### Visual variants

Badges support color palettes to convey semantic meaning or match a theme:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400">
    <Badge colorPalette="primary">Primary</Badge>
    <Badge colorPalette="positive">Positive</Badge>
    <Badge colorPalette="info">Info</Badge>
    <Badge colorPalette="critical">Critical</Badge>
    <Badge colorPalette="cyan">Cyan</Badge>
  </Stack>
)
```

### With icons

Badges can contain icons alongside text for enhanced visual communication:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400">
    <Badge>
      <Icons.SentimentSatisfied />
      Verified
    </Badge>
    <Badge colorPalette="critical">
      <Icons.Error />
      Error
    </Badge>
  </Stack>
)
```
