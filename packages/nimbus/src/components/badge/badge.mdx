# Badge

Badges are small visual indicators used for status displays, notifications, or labels. They're useful for highlighting information, showing counts, or indicating status.

## Basic Usage

A simple Badge displays a short text label with default styling.

```jsx-live
const App = () => <Badge>Default Badge</Badge>
```

## Sizes

Badges come in different sizes to accommodate various UI needs. Available sizes are `2xs`, `xs`, and `md`.

```jsx-live
const App = () => {
  const sizes = ['2xs', 'xs', 'md'];
  
  return (
    <Stack direction="horizontal" gap="400" alignItems="center">
      {sizes.map(size => (
        <Badge key={size} size={size}>{size}</Badge>
      ))}
    </Stack>
  )
}
```

## Colors

Badges support all color palettes. You can use both system colors and semantic colors with the Badge component.

### System Colors

```jsx-live
const App = () => {
  const systemColors = ['tomato', 'red', 'blue', 'cyan', 'teal', 'green', 'grass', 'amber', 'yellow', 'orange'];
  
  return (

      <Stack direction="horizontal" gap="400" alignItems="center" flexWrap="wrap">
        {systemColors.map(color => (
          <Badge key={color} colorPalette={color}>{color}</Badge>
        ))}
      </Stack>
  )
}
```

### Semantic Colors

```jsx-live
const App = () => {
  const semanticColors = ['primary', 'neutral', 'info', 'success', 'error', 'danger'];
  
  return (
    <Stack gap="400">
      <Text fontWeight="bold">Semantic Colors</Text>
      <Stack direction="horizontal" gap="400" alignItems="center" flexWrap="wrap">
        {semanticColors.map(color => (
          <Badge key={color} colorPalette={color}>{color}</Badge>
        ))}
      </Stack>
    </Stack>
  )
}
```

## With Icons

Badges can include icons to enhance visual communication. Icons can be positioned before, after, or on both sides of the text.

```jsx-live
const App = () => {
  const demoText = "Badge";
  
  return (
    <Stack direction="horizontal" gap="400" alignItems="center">
      <Badge>
        {demoText} <Icons.SentimentSatisfied />
      </Badge>
      
      <Badge>
        <Icons.SentimentSatisfied /> {demoText}
      </Badge>
      
      <Badge>
        <Icons.SentimentSatisfied /> {demoText} <Icons.SentimentSatisfied />
      </Badge>
    </Stack>
  )
}
```

## HTML Element Customization

By default, Badge renders as a `<span>` element, but you can change it by using the `as` prop.

```jsx-live
const App = () => (
  <Stack gap="400" direction="row">
    <Badge>Default span</Badge>
    <Badge as="div">As a div</Badge>
  </Stack>
)
```

## Data and ARIA Attributes

The Badge component supports forwarding `data-*` and `aria-*` attributes to the DOM, which can be useful for testing and accessibility.

```jsx-live
const App = () => (
  <Badge 
    data-testid="status-badge" 
    aria-label="Status: Active"
    colorPalette="grass"
  >
    Active
  </Badge>
)
```
