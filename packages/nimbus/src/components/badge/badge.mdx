# Badge

Briefly highlights or categorizes associated UI elements with concise visual
cues for status or metadata.

## Overview

Badges are small, visually distinct labels used to highlight status, category,
or counts associated with other UI elements. They feature concise text and
customizable styling for clear communication without overwhelming the primary
content.

## Variables

Get familiar with the features.

### Size

#### Medium

This is the default size.

```jsx live
const App = () => (
  <Badge colorPalette="primary" size="md">
    <Icons.SentimentSatisfied />
    Badge
    <Icons.SentimentSatisfied />
  </Badge>
)
```

#### X-small

For uses where several badges will be shown at one time.

```jsx live
const App = () => (
  <Badge colorPalette="primary" size="xs">
    <Icons.SentimentSatisfied />
    Badge
    <Icons.SentimentSatisfied />
  </Badge>
)
```

#### 2X-small

Use these when there are a more than a few badges at one time, or to be less
distracting to let the user focus on other parts of the surrounding page.

```jsx live
const App = () => (
  <Badge colorPalette="primary" size="2xs">
    <Icons.SentimentSatisfied />
    Badge
    <Icons.SentimentSatisfied />
  </Badge>
)
```

### Visual options

Badges can be set to use either the semantic color palette or any other radix
color. When pairing new radix colors, the formula uses the 3rd color shade for
the background and the 11th for text to maintain text accessibility.

#### Semantic colors

When a badge is supposed to convey a specific and consistent meaning, use these
predefined color options.

```jsx live
const App = () => {
  const semanticColors = ['info', 'primary', 'positive', 'critical', 'warning', 'neutral'];

  return (
    <Stack direction="horizontal" gap="400" alignItems="center" flexWrap="wrap">
      {semanticColors.map(color => (
        <Badge key={color} colorPalette={color}>{color}</Badge>
      ))}
    </Stack>
  )
}
```

#### Custom colors

When the color of the badge serves a purely aesthetic purpose, or the semantic
color palette does not offer enough options, use any of these available options
from our color palette.

```jsx live
const App = () => {
  const customColors = ['teal', 'pink', 'mint'];

  return (
    <Stack direction="horizontal" gap="400" alignItems="center" flexWrap="wrap">
      {customColors.map(color => (
        <Badge key={color} colorPalette={color}>{color}</Badge>
      ))}
    </Stack>
  )
}
```
