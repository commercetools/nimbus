## Overview

The keyboard key component provides semantic and visual representation for
keyboard input, such as a shortcut, key press, or command line input, ensuring
clarity and consistency when providing user instructions.

## Variables

Get familiar with the features.

### Basic

```jsx live
const App = () => <Kbd>Ctrl</Kbd>
```

### Key combinations

```jsx live
const App = () => <div>Press <Kbd>Ctrl</Kbd> + <Kbd>C</Kbd> to copy</div>
```
