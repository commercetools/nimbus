## Accessibility

Accessibility ensures that digital content and functionality are usable by
everyone, including people with disabilities, by addressing visual, auditory,
cognitive, and physical limitations.

```jsx live
const App = () => (
  <Flex as="nav" gap="400" p="400">
    <Box as="a" href="#" p="200">
      Home
    </Box>
    <Box as="a" href="#" p="200">
      About
    </Box>
    <Box as="a" href="#" p="200">
      Contact
    </Box>
  </Flex>
);
```

### Accessibility standards

- **Use semantic HTML**: Choose appropriate HTML elements with the `as` prop
- **Maintain reading order**: Ensure visual order matches DOM order for screen
  readers
- **Avoid layout-only divs**: When possible, apply flex styles to semantic
  elements
