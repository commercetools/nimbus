# Flex

Flex is a layout component that provides a convenient way to create flexible layouts based on CSS Flexbox. It's based on the [Chakra UI Flex component](https://chakra-ui.com/docs/components/flex) and supports all style props for rapid UI development.

## Overview

The Flex component is essentially a Box component with `display: flex` applied by default. It provides an intuitive API for creating flexible layouts and supports all flexbox properties.

### Key Features

- **Flexbox by default**: Automatically applies `display: flex`
- **Style props**: Supports all Chakra UI style props for rapid development
- **Responsive design**: Easily create responsive layouts with responsive props
- **Semantic HTML**: Can render as any HTML element using the `as` prop

### Resources

Learn more about CSS Flexbox and the underlying implementation:

[Chakra UI Flex](https://chakra-ui.com/docs/components/flex)
[CSS Flexbox Guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)

## Usage

### Import

```jsx
import { Flex } from "@commercetools/nimbus";
```

### Basic usage

```jsx-live
const App = () => (
  <Flex gap="400">
    <Box p="400" bg="primary.7">
      Item 1
    </Box>
    <Box p="400" bg="primary.7">
      Item 2
    </Box>
    <Box p="400" bg="primary.7">
      Item 3
    </Box>
  </Flex>
);
```

### Direction

Control the direction of flex items using the `direction` prop:

```jsx-live
const App = () => (
  <Stack gap="600">
    <Stack gap="200">
      <Text fontWeight="500">Horizontal (row):</Text>
      <Flex direction="row" gap="400">
        <Box p="400" bg="primary.7">
          Item 1
        </Box>
        <Box p="400" bg="primary.7">
          Item 2
        </Box>
        <Box p="400" bg="primary.7">
          Item 3
        </Box>
      </Flex>
    </Stack>
    <Stack gap="200">
      <Text fontWeight="500">Vertical (column):</Text>
      <Flex direction="column" gap="400">
        <Box p="400" bg="primary.7">
          Item 1
        </Box>
        <Box p="400" bg="primary.7">
          Item 2
        </Box>
        <Box p="400" bg="primary.7">
          Item 3
        </Box>
      </Flex>
    </Stack>
  </Stack>
);
```

### Alignment

Control alignment of flex items using `alignItems` and `justifyContent`:

```jsx-live
const App = () => (
  <Flex
    direction="row"
    gap="400"
    alignItems="center"
    justifyContent="space-between"
    p="400"
    bg="neutral.2"
  >
    <Box p="400" bg="primary.7">
      Item 1
    </Box>
    <Box p="400" bg="primary.7">
      Item 2
    </Box>
    <Box p="400" bg="primary.7">
      Item 3
    </Box>
  </Flex>
);
```

### Wrapping

Control how flex items wrap using the `wrap` prop:

```jsx-live
const App = () => (
  <Flex wrap="wrap" gap="400" maxWidth="600px">
    <Box p="400" bg="primary.7" minWidth="150px">
      Item 1
    </Box>
    <Box p="400" bg="primary.7" minWidth="150px">
      Item 2
    </Box>
    <Box p="400" bg="primary.7" minWidth="150px">
      Item 3
    </Box>
    <Box p="400" bg="primary.7" minWidth="150px">
      Item 4
    </Box>
    <Box p="400" bg="primary.7" minWidth="150px">
      Item 5
    </Box>
  </Flex>
);
```

### Flexible items

Control how individual items grow and shrink using flex properties:

```jsx-live
const App = () => (
  <Flex gap="400">
    <Box p="400" bg="primary.7" flex="1">
      Flexible item (flex: 1)
    </Box>
    <Box p="400" bg="positive.7">
      Fixed width item
    </Box>
    <Box p="400" bg="primary.7" flex="2">
      More flexible item (flex: 2)
    </Box>
  </Flex>
);
```

### Centering content

A common use case is centering content both horizontally and vertically:

```jsx-live
const App = () => (
  <Flex
    alignItems="center"
    justifyContent="center"
    height="200px"
    bg="neutral.2"
  >
    <Box p="400" bg="primary.7">
      Centered content
    </Box>
  </Flex>
);
```

### With asChild

Use the `asChild` prop to apply Flex styles to any child element while maintaining the child's functionality:

```jsx-live
const App = () => (
  <Flex gap="400" p="400" bg="neutral.2" asChild>
    <section>
      <Box p="400" bg="primary.7">
        Item 1
      </Box>
      <Box p="400" bg="primary.7">
        Item 2
      </Box>
    </section>
  </Flex>
);
```

## Guidelines

Use the Flex component strategically to create flexible and responsive layouts.

### Best practices

- **Use for flexible layouts**: Flex is ideal when you need items to grow, shrink, or align dynamically
- **Consider Stack first**: For simple vertical or horizontal stacking with consistent spacing, Stack may be simpler
- **Leverage responsive props**: Use responsive values for direction, alignment, and other properties
- **Combine with other layout components**: Flex works well with Box, Stack, and Grid for complex layouts

### When to use

> [!TIP]\
> When to use

- **Dynamic layouts**: When items need to grow or shrink based on available space
- **Complex alignment**: When you need precise control over item alignment
- **Responsive designs**: When layout needs to change based on viewport size
- **Flexible spacing**: When items should distribute space dynamically

> [!CAUTION]\
> When not to use

- **Simple stacking**: Use Stack instead for basic vertical/horizontal layouts with consistent spacing
- **Grid layouts**: Use Grid for two-dimensional layouts with rows and columns
- **Static layouts**: Use Box when flexbox features aren't needed

## Props

The Flex component accepts all system props from Chakra UI, including:

### Flexbox-specific props

- `direction` - Sets the direction of flex items (`row`, `column`, `row-reverse`, `column-reverse`)
- `wrap` - Controls wrapping behavior (`wrap`, `nowrap`, `wrap-reverse`)
- `alignItems` - Aligns items on the cross axis
- `justifyContent` - Aligns items on the main axis
- `alignContent` - Aligns flex lines when there's extra space
- `gap` - Sets the gap between items (replaces the need for margins)

### All Box props

Flex also supports all Box component props:

- Space props (margin, padding)
- Color props (background, color)
- Typography props (fontSize, fontWeight)
- Layout props (width, height, display)
- Border props
- Position props
- Shadow props

## Accessibility

Flex renders a `div` by default but can be changed to any other HTML element using the `as` prop to ensure proper semantic structure:

```jsx-live
const App = () => (
  <Flex as="nav" gap="400" p="400">
    <Box as="a" href="#" p="200">
      Home
    </Box>
    <Box as="a" href="#" p="200">
      About
    </Box>
    <Box as="a" href="#" p="200">
      Contact
    </Box>
  </Flex>
);
```

### Best practices

- **Use semantic HTML**: Choose appropriate HTML elements with the `as` prop
- **Maintain reading order**: Ensure visual order matches DOM order for screen readers
- **Avoid layout-only divs**: When possible, apply flex styles to semantic elements
