## Overview

The money input component combines a currency selector (a dropdown menu) and a
numerical input field into a single, cohesive unit, offering robust states like
default, invalid, and disabled, alongside specialized handling for focused and
hovered interactions.

### Anatomy

#### Multi-part functions

In each localized money input, there is a dropdown to select the correct
currency, a field that lets a user type in directly, and a stepper to allow for
incremental numerical changes to the price shown within.

```jsx live
const App = () => {
  const [value, setValue] = React.useState({
    amount: "20.00",
    currencyCode: "EUR"
  });

  return (
    <MoneyInput
      value={value}
      onValueChange={setValue}
      currencies={["EUR", "USD", "GBP", "JPY"]}
    />
  );
}
```

#### High-precision pricing

This mode is essential for maintaining financial accuracy in large-scale or
complex commercial systems.

```jsx live
const App = () => {
  const [value, setValue] = React.useState({
    amount: "20.12345",
    currencyCode: "EUR"
  });

  return (
    <MoneyInput
      value={value}
      onValueChange={setValue}
      currencies={["EUR", "USD", "GBP"]}
      hasHighPrecisionBadge
    />
  );
}
```
