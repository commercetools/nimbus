## Variables

Get familiar with the features.

### Size

**Medium** (40px)

Most commonly used in the upper right hand side of the page paired with a
drop-down menu.

```jsx live
const App = () => (
  <Stack direction="horizontal">
    <Avatar firstName="Adam" lastName="Vadam" src="https://thispersondoesnotexist.com/ " size="md" />
    <Avatar firstName="Adam" lastName="Vadam" size="md" /> {/* Shows initials when no image */}
  </Stack>
)
```

**X-Small** (32px)

Use small circle avatars in tighter areas like text fields and dropdown fields
or when it is less important information.

```jsx live
const App = () => (
  <Stack direction="horizontal">
    <Avatar firstName="Adam" lastName="Vadam" src="https://thispersondoesnotexist.com/ " size="xs" />
    <Avatar firstName="Adam" lastName="Vadam" size="xs" /> {/* Shows initials when no image */}
  </Stack>
)
```

**2x-Small** (24px)

Use these in condensed areas like notifications or filters.

```jsx live
const App = () => (
  <Stack direction="horizontal">
    <Avatar firstName="Adam" lastName="Vadam" src="https://thispersondoesnotexist.com/ " size="2xs" />
    <Avatar firstName="Adam" lastName="Vadam" size="2xs" /> {/* Shows initials when no image */}
  </Stack>
)
```

### Initials and images

#### Avatar initials

The default and fallback for avatar graphics is to use initials. Default color
should use Primary, but is open to custom colors.

```jsx live
const App = () => (
  <Avatar firstName="Adam" lastName="Vadam" />
)
```
