# RadioInput

A RadioInput component allows users to select a single option from a mutually exclusive set of choices.

## Basic Usage

[Explain the basic usage / usecase of the component].

```jsx-live
const App = () => (
  <RadioInputGro.Root name="example">
    <Stack direction="horizontal" gap="600">
      <RadioInput.Option value="option1">Option 1</RadioInput.Option>
      <RadioInput.Option value="option2">Option 2</RadioInput.Option>
      <RadioInput.Option value="option3">Option 3</RadioInput.Option>
    </Stack>
  </RadioInputGro.Root>
)
```


### Sizes

Available sizes.

```jsx-live
const App = () => {

  const sizes = ['2xl', 'xl', 'lg', 'md', 'sm', 'xs', '2xs'];

  return (
    <Stack direction="horizontal" alignItems="center">
      {sizes.map(size => (
```

### Variants

Available variants.

```jsx-live
const App = () => {
```
