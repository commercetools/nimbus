## Overview

Radio inputs ensure users can easily select one option from a small, visible set
of mutually exclusive choices with clear labels and visual feedback.

### Resources

Deep dive into implementation details and access the Nimbus design library.

### Visual options

#### Default

```jsx live
const App = () => (
  <RadioInput.Root orientation="horizontal" name="default-example">
    <RadioInput.Option value="option1">Select me</RadioInput.Option>
    <RadioInput.Option value="option2">Select me instead</RadioInput.Option>
  </RadioInput.Root>
)
```

#### Active

```jsx live
const App = () => (
  <RadioInput.Root orientation="horizontal" name="active-example" defaultValue="option1">
    <RadioInput.Option value="option1">Selected option</RadioInput.Option>
    <RadioInput.Option value="option2">Unselected option</RadioInput.Option>
  </RadioInput.Root>
)
```

#### Disabled

```jsx live
const App = () => (
  <RadioInput.Root orientation="horizontal" name="disabled-example" isDisabled>
    <RadioInput.Option value="option1">Disabled option</RadioInput.Option>
    <RadioInput.Option value="option2">Also disabled</RadioInput.Option>
  </RadioInput.Root>
)
```

#### Invalid toned

```jsx live
const App = () => (
  <RadioInput.Root orientation="horizontal" name="invalid-example" isInvalid>
    <RadioInput.Option value="option1">Invalid option</RadioInput.Option>
    <RadioInput.Option value="option2">Another invalid option</RadioInput.Option>
  </RadioInput.Root>
)
```
