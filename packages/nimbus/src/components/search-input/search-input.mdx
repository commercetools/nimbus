## Overview

Search inputs provide a dedicated text field for users to enter keywords or phrases to query and retrieve matching data or content. They are essential for navigating large amounts of information.

## Variables

Get familiar with the features.

### Visual Options

#### Size

The medium size of the SearchInput is default, the small size is meant for more compact uses.

```jsx live
const App = () => (
  <Stack direction="row" gap="400" alignItems="center">
    <SearchInput size="md" placeholder="Search..." />
    <SearchInput size="sm" placeholder="Search..." />
  </Stack>
)
```

### Appearance

Allows switching the visual style between solid and ghost appearance.

```jsx live
const App = () => (
  <Stack direction="row" gap="400" alignItems="center">
    <SearchInput variant="solid" placeholder="Search..." />
    <SearchInput variant="ghost" placeholder="Search..." />
  </Stack>
)
```
