# Icon Component

Icon is a wrapper component for displaying SVG icons from the Nimbus icon library. It provides consistent sizing, coloring, and styling for icons throughout your application.

## Getting started

### Import

```tsx
import { Icon, type IconProps } from "@commercetools/nimbus";
import { CheckCircle, Settings, Delete } from "@commercetools/nimbus-icons";
```

> **Note**: Icons are imported separately from `@commercetools/nimbus-icons`, which provides a comprehensive set of Material Design icons and custom Nimbus icons.

### Basic usage

Icons can be used in two ways: by passing the icon component as a child, or using the `as` prop as shorthand:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400" align="center">
    {/* Using children */}
    <Icon>
      <CheckCircle />
    </Icon>
    
    {/* Using as prop (shorthand) */}
    <Icon as={Settings} />
  </Stack>
)
```

## Usage examples

### Size variants

Icon provides six predefined size variants that can be applied using the `size` prop:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400" align="center">
    <Stack direction="column" gap="100" align="center">
      <Icon size="2xs" as={Star} />
      <Text fontSize="xs">2xs</Text>
    </Stack>
    <Stack direction="column" gap="100" align="center">
      <Icon size="xs" as={Star} />
      <Text fontSize="xs">xs</Text>
    </Stack>
    <Stack direction="column" gap="100" align="center">
      <Icon size="sm" as={Star} />
      <Text fontSize="xs">sm</Text>
    </Stack>
    <Stack direction="column" gap="100" align="center">
      <Icon size="md" as={Star} />
      <Text fontSize="xs">md</Text>
    </Stack>
    <Stack direction="column" gap="100" align="center">
      <Icon size="lg" as={Star} />
      <Text fontSize="xs">lg</Text>
    </Stack>
    <Stack direction="column" gap="100" align="center">
      <Icon size="xl" as={Star} />
      <Text fontSize="xs">xl</Text>
    </Stack>
  </Stack>
)
```

### Custom size

For sizes beyond the predefined variants, use the `boxSize` style prop:

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400" align="center">
    <Icon boxSize="1600" as={Favorite} />
    <Icon boxSize="2400" as={Favorite} />
    <Icon boxSize="3200" as={Favorite} />
  </Stack>
)
```

### Color customization

Icons inherit the current text color by default, but can be customized using the `color` prop with Nimbus color tokens or any valid CSS color:

```jsx-live-dev
const App = () => (
  <Stack direction="column" gap="400">
    <Stack direction="row" gap="300" align="center">
      <Icon as={Info} color="primary.11" size="lg" />
      <Icon as={CheckCircle} color="positive.11" size="lg" />
      <Icon as={Warning} color="warning.11" size="lg" />
      <Icon as={Error} color="critical.11" size="lg" />
      <Icon as={Help} color="info.11" size="lg" />
    </Stack>
    
    <Stack direction="row" gap="300" align="center">
      <Icon as={Favorite} color="deeppink" size="lg" />
      <Icon as={Star} color="gold" size="lg" />
      <Icon as={Palette} color="rebeccapurple" size="lg" />
    </Stack>
  </Stack>
)
```

### Using the `as` prop

The `as` prop provides a convenient shorthand for rendering an icon without nesting:

```jsx-live-dev
const App = () => {
  const [iconType, setIconType] = React.useState<IconProps["as"]>(Home);
  
  return (
    <Stack direction="column" gap="400">
      <Stack direction="row" gap="200">
        <Button size="xs" onClick={() => setIconType(Home)}>Home</Button>
        <Button size="xs" onClick={() => setIconType(Settings)}>Settings</Button>
        <Button size="xs" onClick={() => setIconType(Notifications)}>Notifications</Button>
        <Button size="xs" onClick={() => setIconType(AccountCircle)}>Account</Button>
      </Stack>
      
      <Stack direction="row" gap="300" align="center">
        <Icon as={iconType} size="xl" color="primary.11" />
        <Text>Selected icon changes dynamically</Text>
      </Stack>
    </Stack>
  );
}
```

### With custom SVG

Icon can also wrap custom SVG elements or components:

```jsx-live-dev
const App = () => {
  const CustomSvg = () => (
    <svg viewBox="0 0 24 24" fill="currentColor">
      <path d="M12 2L2 7l10 5 10-5-10-5zM2 17l10 5 10-5M2 12l10 5 10-5" />
    </svg>
  );
  
  return (
    <Stack direction="row" gap="400" align="center">
      <Icon size="sm">
        <CustomSvg />
      </Icon>
      <Icon size="md" color="primary.11">
        <CustomSvg />
      </Icon>
      <Icon size="lg" color="positive.11">
        <CustomSvg />
      </Icon>
    </Stack>
  );
}
```

### Responsive sizing

Icon supports responsive prop values for different breakpoints:

```jsx-live-dev
const App = () => (
  <Stack direction="column" gap="400">
    <Icon 
      as={Smartphone}
      size={{ base: "sm", md: "md", lg: "xl" }}
      color="primary.11"
    />
    <Text fontSize="sm" color="neutral.11">
      Icon size changes from sm → md → xl as viewport increases
    </Text>
  </Stack>
)
```

### Styling with Chakra props

Icon accepts all Chakra UI SVG props for advanced styling:

```jsx-live-dev

  <Stack direction="row" gap="400" align="center">
    <Icon 
      as={FavoriteBorder}
      size="xl"
      color="critical.11"
      opacity="0.7"
      _hover={{ opacity: 1, transform: "scale(1.1)" }}
      transition="all 0.2s"
      cursor="pointer"
    />

      as={Star}
      size="xl"
      color="warning.11"
      filter="drop-shadow(0 2px 4px rgba(0,0,0,0.2))"
    />
    <Icon 
      as={Lightbulb}
      size="xl"
      color="info.11"
      transform="rotate(15deg)"
    />
  </Stack>
)
```

## Component requirements

### Usage context

Icon should be used when you need to:
- Display visual symbols from the Nimbus icon library
- Add icons to buttons, menus, lists, or other UI elements
- Create status indicators or visual cues
- Maintain consistent icon sizing and styling

Icons from `@commercetools/nimbus-icons` include:
- **Material Design icons**: Comprehensive set of standard UI icons
- **Custom Nimbus icons**: Brand-specific and specialized icons (Figma, Github, CommercetoolsCube, etc.)

### Accessibility

Icon is a visual element that requires proper accessibility considerations:

#### ARIA Attributes

- Use `aria-label` for standalone icons that convey meaning
- Use `aria-hidden="true"` for decorative icons next to text labels
- Ensure sufficient color contrast (avoid relying solely on color)

```tsx
{/* Standalone icon - needs label */}
<IconButton aria-label="Delete item" icon={Delete} />

{/* Decorative icon with text - hide from screen readers */}
<Button>
  <Icon as={Save} aria-hidden="true" />
  Save Document
</Button>
```

#### Keyboard Navigation

Icons themselves are not interactive. When icons need to be clickable:
- Wrap them in Button or IconButton components
- Ensure the parent element is keyboard accessible
- Provide appropriate focus indicators

#### Persistent ID

If your use case requires tracking and analytics for this component, it is good practice to add a **persistent**, **unique** id to the component:

```tsx
<Icon id="status-icon-success" as={CheckCircle} />
```

## API reference

### `children`

Accepts a single icon component from `@commercetools/nimbus-icons` or a custom SVG element.

```tsx
<Icon>
  <CheckCircle />
</Icon>
```

### `as`

Shorthand prop that accepts an icon component to be rendered.

```tsx
<Icon as={CheckCircle} />
```

### `size`

Predefined size variant for the icon.

- **Values**: `"2xs"` | `"xs"` | `"sm"` | `"md"` | `"lg"` | `"xl"`
- **Default**: Inherits font-size from parent

```tsx
<Icon as={Star} size="md" />
```

### `color`

Sets the icon color using Nimbus color tokens or CSS color values.

```tsx
<Icon as={Info} color="primary.11" />
<Icon as={Warning} color="gold" />
```

### `boxSize`

Custom size using Nimbus size tokens or CSS values.

```tsx
<Icon as={Settings} boxSize="2000" />
<Icon as={Home} boxSize="48px" />
```

### Inherited Props

Icon extends Chakra UI's SVG element and inherits all [Chakra UI style props](https://www.chakra-ui.com/docs/styled-system/style-props), including opacity, transform, filter, cursor, transitions, and responsive values.

## Common patterns

### Icons in buttons

Combine icons with buttons for enhanced visual communication:

```jsx-live-dev
const App = () => (
  <Stack direction="column" gap="400">
    <Stack direction="row" gap="200">
      <Button size="xs" variant="solid" colorPalette="primary">
        <Icon as={Add} />
        Create New
      </Button>
      <Button size="xs" variant="outline">
        <Icon as={Save} />
        Save
      </Button>
      <Button size="xs" variant="ghost">
        <Icon as={Delete} />
        Delete
      </Button>
    </Stack>
    
    <Stack direction="row" gap="200">
      <IconButton icon={Edit} aria-label="Edit" size="xs" />
      <IconButton icon={Share} aria-label="Share" size="xs" />
      <IconButton icon={MoreVert} aria-label="More options" size="xs" />
    </Stack>
  </Stack>
)
```

### Navigation icons

Create icon-based navigation elements:

```jsx-live-dev
const App = () => {
  const [selected, setSelected] = React.useState("home");
  
  const NavItem = ({ id, icon, label }) => (
    <Stack 
      direction="column" 
      gap="100" 
      align="center"
      p="300"
      borderRadius="200"
      bg={selected === id ? "primary.3" : "transparent"}
      color={selected === id ? "primary.11" : "neutral.11"}
      cursor="pointer"
      onClick={() => setSelected(id)}
      _hover={{ bg: selected === id ? "primary.3" : "neutral.3" }}
    >
      <Icon as={icon} size="md" />
      <Text fontSize="xs">{label}</Text>
    </Stack>
  );
  
  return (
    <Stack direction="row" gap="200" p="400" bg="neutral.2" borderRadius="200">
      <NavItem id="home" icon={Home} label="Home" />
      <NavItem id="search" icon={Search} label="Search" />
      <NavItem id="notifications" icon={Notifications} label="Alerts" />
      <NavItem id="settings" icon={Settings} label="Settings" />
    </Stack>
  );
}
```

### Icon lists

Display lists with icon indicators:

```jsx-live-dev
const App = () => {
  const features = [
    { icon: CheckCircle, text: "Real-time synchronization", color: "positive.11" },
    { icon: Security, text: "Enterprise-grade security", color: "info.11" },
    { icon: Speed, text: "Lightning-fast performance", color: "primary.11" },
    { icon: CloudDone, text: "Cloud-based storage", color: "info.11" },
  ];
  
  return (
    <Stack direction="column" gap="300">
      {features.map((feature, index) => (
        <Stack key={index} direction="row" gap="300" align="center">
          <Icon as={feature.icon} color={feature.color} size="sm" />
          <Text>{feature.text}</Text>
        </Stack>
      ))}
    </Stack>
  );
}
```

### File type indicators

Use icons to represent different file types or content:

```jsx-live-dev
const App = () => {
  const files = [
    { name: "Document.pdf", icon: PictureAsPdf, color: "critical.11" },
    { name: "Spreadsheet.xlsx", icon: TableChart, color: "positive.11" },
    { name: "Presentation.pptx", icon: Slideshow, color: "warning.11" },
    { name: "Image.jpg", icon: Image, color: "info.11" },
    { name: "Archive.zip", icon: FolderZip, color: "neutral.11" },
  ];
  
  return (
    <Stack direction="column" gap="200">
      {files.map((file, index) => (
        <Stack 
          key={index} 
          direction="row" 
          gap="300" 
          align="center"
          p="300"
          borderRadius="200"
          _hover={{ bg: "neutral.3" }}
        >
          <Icon as={file.icon} color={file.color} size="md" />
          <Text>{file.name}</Text>
        </Stack>
      ))}
    </Stack>
  );
}
```
