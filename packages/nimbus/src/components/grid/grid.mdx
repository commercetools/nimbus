## Basic usage

Use Grid â†’ When you need fine control over layout, spanning, and custom grid
behaviors.

```jsx
import { Grid } from "@commercetools/nimbus";
```

To use the Grid layout, ensure to wrap the individual grid items inside of the
`<Grid.Item>` component

```jsx live
const App = () => (
  <Grid gap="100">
    <Grid.Item>
      <Box key="1" p="400" bg="neutral.7">
        Item 1
      </Box>
    </Grid.Item>
    <Grid.Item>
      <Box key="1" p="400" bg="neutral.7">
        Item 2
      </Box>
    </Grid.Item>
  </Grid>
)
```

### With columns

```jsx live
    <Grid.Item>

```

### With varying layout

Showcases how elements can span multiple rows and columns within a structured
grid. This layout is useful for dashboards, content cards, or complex UI
components where elements need different sizes while maintaining alignment.

```jsx live
const App = () => (
  <Grid
    h="200px"
    templateRows="repeat(2, 1fr)"
    templateColumns="repeat(5, 1fr)"
    gap={4}
  >
    <Grid.Item rowSpan={2} colSpan={1}>
      <Box p="400" bg="neutral.7">
        colSpan=1
      </Box>
    </Grid.Item>
    <Grid.Item colSpan={2}>
      <Box p="400" bg="neutral.7">
        colSpan=2
      </Box>
    </Grid.Item>
    <Grid.Item colSpan={2}>
      <Box p="400" bg="neutral.7">
        colSpan=2
      </Box>
    </Grid.Item>
    <Grid.Item colSpan={4}>
      <Box p="400" bg="neutral.7">
        colSpan=4
      </Box>
    </Grid.Item>
  </Grid>
)
```

### With template areas properties

Demonstrate layout with template areas - shows how to create a common page
layout pattern

```jsx live
const App = () => (
 <Grid
    h="400px"
    templateAreas={`
      "header header"
      "nav main"
      "footer footer"
    `}
    templateColumns="200px 1fr"
    gap="400"
  >
    <Grid.Item gridArea="header">
      <Box p="400" bg="neutral.7" h="full">
        Header
      </Box>
    </Grid.Item>
    <Grid.Item gridArea="nav">
      <Box p="400" bg="neutral.7" h="full">
        Navigation
      </Box>
    </Grid.Item>
    <Grid.Item gridArea="main">
      <Box p="400" bg="neutral.7" h="full">
        Main Content
      </Box>
    </Grid.Item>
    <Grid.Item gridArea="footer">
      <Box p="400" bg="neutral.7" h="full">
        Footer
      </Box>
    </Grid.Item>
  </Grid>
)
```
