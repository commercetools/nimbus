## Getting started

### Import

```tsx
import { Toolbar, type ToolbarProps } from '@commercetools/nimbus';
```

### Basic usage

The simplest implementation creates a horizontal toolbar with default styling:

```jsx-live-dev
const App = () => (
  <Toolbar>
    <Button variant="ghost">File</Button>
    <Button variant="ghost">Edit</Button>
    <Button variant="ghost">View</Button>
  </Toolbar>
)
```

### Orientation

The `orientation` prop controls the layout direction and keyboard navigation:

```jsx-live-dev
const App = () => (
  <Stack direction="column" gap="600" alignItems="flex-start">
    <Box>
      <Text textStyle="sm" mb="200" color="neutral.11">Horizontal (default)</Text>
      <Toolbar orientation="horizontal">
        <IconButton size="xs" variant="ghost" aria-label="Home">
          <Icons.Home />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Search">
          <Icons.Search />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Settings">
          <Icons.Settings />
        </IconButton>
      </Toolbar>
    </Box>
    <Box>
      <Text textStyle="sm" mb="200" color="neutral.11">Vertical</Text>
      <Toolbar orientation="vertical">
        <IconButton size="xs" variant="ghost" aria-label="Home">
          <Icons.Home />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Search">
          <Icons.Search />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Settings">
          <Icons.Settings />
        </IconButton>
      </Toolbar>
    </Box>
  </Stack>
)
```

**Behavioral differences:**
- `orientation="horizontal"`: Arrow Left/Right keys navigate between controls
- `orientation="vertical"`: Arrow Up/Down keys navigate between controls

## Common patterns

### Text editor toolbar

Create a comprehensive text formatting toolbar with multiple control groups:

```jsx-live-dev
const App = () => {
  const [textStyle, setTextStyle] = useState("p");

  const textStyles = [
    { id: "h1", label: "Heading 1" },
    { id: "h2", label: "Heading 2" },
    { id: "p", label: "Paragraph" },
  ];

  const selectedTextStyle = textStyles.find((v) => v.id === textStyle);

  return (
    <Toolbar size="md" variant="outline" aria-label="Text formatting">
      <Group>
        <Menu.Root onAction={(v) => setTextStyle(String(v))}>
          <Menu.Trigger
            borderRadius="200"
            border="1px solid"
            borderColor="neutral.6"
            width="3500"
            aria-label="Text style"
          >
            <Box display="flex" alignItems="center" gap="200" px="200">
              <Text>{selectedTextStyle?.label}</Text>
              <Icons.KeyboardArrowDown />
            </Box>
          </Menu.Trigger>
          <Menu.Content>
            {textStyles.map((style) => (
              <Menu.Item key={style.id} id={style.id}>
                {style.label}
              </Menu.Item>
            ))}
          </Menu.Content>
        </Menu.Root>
      </Group>
      <Separator orientation="vertical" />
      <ToggleButtonGroup.Root
        selectionMode="multiple"
        defaultSelectedKeys={["bold"]}
        aria-label="Text formatting"
      >
        <IconToggleButton id="bold" size="xs" variant="ghost" aria-label="Bold">
          <Icons.FormatBold />
        </IconToggleButton>
        <IconToggleButton id="italic" size="xs" variant="ghost" aria-label="Italic">
          <Icons.FormatItalic />
        </IconToggleButton>
        <IconToggleButton id="underline" size="xs" variant="ghost" aria-label="Underline">
          <Icons.FormatUnderlined />
        </IconToggleButton>
      </ToggleButtonGroup.Root>
      <Separator orientation="vertical" />
      <ToggleButtonGroup.Root
        selectionMode="single"
        defaultSelectedKeys={["left"]}
        aria-label="Text alignment"
      >
        <IconToggleButton id="left" size="xs" variant="ghost" aria-label="Align Left">
          <Icons.FormatAlignLeft />
        </IconToggleButton>
        <IconToggleButton id="center" size="xs" variant="ghost" aria-label="Align Center">
          <Icons.FormatAlignCenter />
        </IconToggleButton>
        <IconToggleButton id="right" size="xs" variant="ghost" aria-label="Align Right">
          <Icons.FormatAlignRight />
        </IconToggleButton>
      </ToggleButtonGroup.Root>
    </Toolbar>
  );
}
```

### Action toolbar

Create a simple action toolbar with primary controls:

```jsx-live-dev
const App = () => (
  <Stack direction="column" gap="400" width="full">
    <Toolbar aria-label="Document actions">
      <Group>
        <IconButton size="xs" variant="ghost" aria-label="New document">
          <Icons.Add />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Open document">
          <Icons.FolderOpen />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Save document">
          <Icons.Save />
        </IconButton>
      </Group>
      <Separator orientation="vertical" />
      <Group>
        <IconButton size="xs" variant="ghost" aria-label="Undo">
          <Icons.Undo />
        </IconButton>
        <IconButton size="xs" variant="ghost" aria-label="Redo" isDisabled>
          <Icons.Redo />
        </IconButton>
      </Group>
    </Toolbar>
    <Box
      p="600"
      borderRadius="200"
      borderWidth="1px"
      borderColor="neutral.6"
      minHeight="6000"
    >
      <Text color="neutral.11">Document content area</Text>
    </Box>
  </Stack>
)
```
