# IconToggleButton

A convenience wrapper around the [ToggleButton](/components/toggle-button) 
component designed for icon-only usage. It provides the same toggle 
functionality with optimizations for icon-based interactions.

## Overview

IconToggleButton maintains all the functionality of ToggleButton while 
enforcing accessibility best practices and providing appropriate styling for 
icon-only toggle buttons.

### Key differences from ToggleButton

- **Required `aria-label`:** Ensures screen reader compatibility for icon-only 
  content
- **Icon Optimized Styling:** Square aspect ratio with zero padding for clean 
  icon presentation
- **Identical API:** All ToggleButton props are supported except 
  `aria-labelledby`

### When to use

Use IconToggleButton for toggle actions that only contain an icon:
- Favorite/unfavorite actions
- Bookmark/unbookmark functions  
- Show/hide content toggles
- Feature enable/disable controls

For toggle buttons with text content, use the standard
[ToggleButton](/components/toggle-button) component.

## Example

```jsx-live
const App = () => (
  <IconToggleButton aria-label="Add to favorites">
    <Icons.Star />
  </IconToggleButton>
);
```
