# Separator

A divider provides a visual separator between sections of content, improving
readability and creating clear boundaries in layouts.

## Overview

Separators are fundamental UI elements used to break up content, improve
readability, and guide the user's eye. They provide semantic meaning through
proper accessibility attributes while maintaining consistent visual styling.

### Basic Usage

The simplest use case is a horizontal divider to separate content blocks:

```jsx-live
const App = () => (
  <Box>
    <Box p="4">
      Content above the divider
    </Box>
    <Separator />
    <Box p="4">
      Content below the divider
    </Box>
  </Box>
)
```

### Vertical Separators

Use vertical dividers to separate side-by-side content:

```jsx-live
const App = () => (
  <Stack direction="row" alignItems="center" gap="4">
    <Box p="4">
      Left content
    </Box>
    <Separator orientation="vertical" />
    <Box p="4">
      Right content
    </Box>
  </Stack>
)
```

### Custom Styling

You can customize the divider using any Chakra UI styling props:

```jsx-live
const App = () => (
  <Stack gap="6">
    <Box>
      <Text fontSize="sm" color="gray.600">Default</Text>
      <Separator />
    </Box>
    <Box>
      <Text fontSize="sm" color="gray.600">Custom color</Text>
      <Separator borderColor="red.300" />
    </Box>
    <Box>
      <Text fontSize="sm" color="gray.600">Thicker divider</Text>
      <Separator borderTopWidth="2px" />
    </Box>
    <Box>
      <Text fontSize="sm" color="gray.600">Dashed style</Text>
      <Separator borderStyle="dashed" borderColor="blue.400" />
    </Box>
  </Stack>
)
```

## Common Use Cases

### Form Sections

Use dividers to separate logical sections within forms:

```jsx-live
const App = () => (
  <Box maxWidth="md">
    <Box p="4">
      <Heading size="md">Personal Information</Heading>
      <Input placeholder="Name" mt="2" />
      <Input placeholder="Email" mt="2" />
    </Box>
    <Separator />
    <Box p="4">
      <Heading size="md">Account Settings</Heading>
      <Input placeholder="Password" type="password" mt="2" />
      <Input placeholder="Confirm Password" type="password" mt="2" />
    </Box>
  </Box>
)
```

### Menu Separators

Group related menu items with dividers:

```jsx-live
const App = () => (
  <Box maxWidth="xs" border="1px" borderColor="gray.200" borderRadius="md">
    <Box p="3" cursor="pointer" _hover={{ bg: "gray.50" }}>New File</Box>
    <Box p="3" cursor="pointer" _hover={{ bg: "gray.50" }}>Open File</Box>
    <Separator />
    <Box p="3" cursor="pointer" _hover={{ bg: "gray.50" }}>Save</Box>
    <Box p="3" cursor="pointer" _hover={{ bg: "gray.50" }}>Save As...</Box>
    <Separator />
    <Box p="3" cursor="pointer" _hover={{ bg: "gray.50" }} color="red.600">Delete</Box>
  </Box>
)
```

### Layout Separators

Create visual boundaries in complex layouts:

```jsx-live
const App = () => (
  <Stack direction="row" height="200px">
    <Box p="4" bg="blue.50" minWidth="200px">
      Sidebar
    </Box>
    <Separator orientation="vertical" />
    <Box p="4" bg="green.50" flex="1">
      Main content area
    </Box>
  </Stack>
)
```

## Accessibility

- The divider automatically receives `role="separator"` for screen readers
- Proper `aria-orientation` is set based on the orientation prop
- Separators are not focusable by default, maintaining proper tab order
- Semantic meaning is preserved for assistive technologies

## API Reference

### Props

| Name          | Type                         | Default        | Description                    |
| ------------- | ---------------------------- | -------------- | ------------------------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the divider |

The component also accepts all standard Chakra UI styling props for
customization (e.g., `borderColor`, `borderWidth`, `borderStyle`, etc.).

_This component is built with React Aria for accessibility and uses Chakra UI's
styling system._
