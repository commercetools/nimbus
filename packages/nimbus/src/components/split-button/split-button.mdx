## Overview

A split button merges a main action that is immediate with a secondary dropdown
menu that holds related actions that are used less often. This component
provides quick access to essential functions while offering a complete list of
options, all while saving space.

## Variables

Get familiar with the features.

### Visuals

#### Solid

Most common styling of the split button. Try to keep only one solid primary
toned button in an area for user clarity for the most important task. Too many
in an area can cause confusion.

```jsx live
const App = () => {
  return (
    <SplitButton
      variant="solid"
      tone="primary"
      onAction={(actionId) => alert(`Action: ${actionId}`)}
      aria-label="More publish options"
    >
      <Menu.Item id="publish">Publish</Menu.Item>
      <Menu.Item id="publish-next">Publish and open next</Menu.Item>
      <Menu.Item id="save">Save as draft</Menu.Item>
    </SplitButton>
  );
}
```

smaller sizes within places like toolbars.

### Size

There are three sizes to split buttons, the default size is medium (md), extra
small (xs), and 2x-small (2xs).

```jsx live
const App = () => (
  <Stack direction="horizontal" gap="400" alignItems="center">
    <SplitButton tone="primary" size="md" onAction={(id) => {}} aria-label="publish options">
      <Menu.Item id="publish">Publish</Menu.Item>
      <Menu.Item id="publish-next">Publish and open next</Menu.Item>
      <Menu.Item id="save">Save as draft</Menu.Item>
    </SplitButton>
    <SplitButton tone="primary" size="xs" onAction={(id) => {}} aria-label="publish options">
      <Menu.Item id="publish">Publish</Menu.Item>
      <Menu.Item id="publish-next">Publish and open next</Menu.Item>
      <Menu.Item id="save">Save as draft</Menu.Item>
    </SplitButton>
    <SplitButton tone="primary" size="2xs" onAction={(id) => {}} aria-label="publish options">
      <Menu.Item id="publish">Publish</Menu.Item>
      <Menu.Item id="publish-next">Publish and open next</Menu.Item>
      <Menu.Item id="save">Save as draft</Menu.Item>
    </SplitButton>
  </Stack>
)
```
