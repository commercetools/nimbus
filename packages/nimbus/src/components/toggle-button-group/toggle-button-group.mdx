## Overview

Button groups can be used several ways, with and without icons, and spanning
from two selections to several.

## Variables

Get familiar with the features.

### Size

Toggle Button Groups come in different sizes to fit various layout needs.

#### Medium

This is the default size.

```jsx live
const App = () => (
  <ToggleButtonGroup.Root aria-label="Medium size example (default)" selectedKeys={["md-day"]} colorPalette="primary">
    <ToggleButtonGroup.Button id="md-day"><Icons.ViewDay />Day</ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="md-week"><Icons.CalendarViewWeek />Week</ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="md-month"><Icons.CalendarViewMonth />Month</ToggleButtonGroup.Button>
  </ToggleButtonGroup.Root>
);
```

#### X-Small

This size is for compact placements.

```jsx live
const App = () => (
  <ToggleButtonGroup.Root size="xs" aria-label="X-Small size example" selectedKeys={["xs-day"]} colorPalette="primary">
    <ToggleButtonGroup.Button id="xs-day"><Icons.ViewDay />Day</ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="xs-week"><Icons.CalendarViewWeek />Week</ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="xs-month"><Icons.CalendarViewMonth />Month</ToggleButtonGroup.Button>
  </ToggleButtonGroup.Root>
);
```

### Visual options

Toggle Button Groups can be configured with labels, icons, or both.

#### Labels and icons

Button groups with icons and labels are used to present a set of related actions
in a compact and visually clear way, where icons provide quick recognition and
labels offer explicit understanding of each option. This combination enhances
user efficiency by making common choices readily available and easily
identifiable.

```jsx live
const App = () => (
  <ToggleButtonGroup.Root aria-label="Labels and icons example" selectedKeys={["li-shipped"]} colorPalette="primary">
    <ToggleButtonGroup.Button id="li-processing">
      <Icons.RotateRight /> Processing
    </ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="li-shipped">
      <Icons.LocalShipping /> Shipped
    </ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="li-cancelled">
      <Icons.Dangerous /> Cancelled
    </ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="li-refunded">
      <Icons.CurrencyExchange /> Refunded
    </ToggleButtonGroup.Button>
  </ToggleButtonGroup.Root>
);
```

#### Just labels

Button groups with just labels provide explicit understanding of each option.
This approach prioritizes readability and ease of comprehension, making it
suitable for scenarios where icons might be ambiguous or when the actions are
best described through text.

```jsx live
const App = () => (
  <ToggleButtonGroup.Root aria-label="Just labels example" selectedKeys={["label-inactive"]} colorPalette="primary">
    <ToggleButtonGroup.Button id="label-active">
      Active promotions
    </ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="label-inactive">
      Inactive promotions
    </ToggleButtonGroup.Button>
    <ToggleButtonGroup.Button id="label-draft">
      Draft promotions
    </ToggleButtonGroup.Button>
  </ToggleButtonGroup.Root>
);
```

#### Just icons

Messages potential problems or situations that require user attention to prevent
issues. They require the user to take action to make needed corrections or
confirmation. Use tooltips for accessibility.

```jsx live
const App = () => (
  <ToggleButtonGroup.Root aria-label="Just icons example" selectedKeys={["icon-copy"]} colorPalette="primary">
    <Tooltip.Root delay={0} closeDelay={0}>
      <ToggleButtonGroup.Button id="icon-edit" aria-label="Edit view">
        <Icons.Edit />
      </ToggleButtonGroup.Button>
      <Tooltip.Content>Edit</Tooltip.Content>
    </Tooltip.Root>
    <Tooltip.Root delay={0} closeDelay={0}>
      <ToggleButtonGroup.Button id="icon-copy" aria-label="Copy view">
        <Icons.ContentCopy />
      </ToggleButtonGroup.Button>
      <Tooltip.Content>Copy</Tooltip.Content>
    </Tooltip.Root>
    <Tooltip.Root delay={0} closeDelay={0}>
      <ToggleButtonGroup.Button id="icon-delete" aria-label="Delete view">
        <Icons.Delete />
      </ToggleButtonGroup.Button>
      <Tooltip.Content>Delete</Tooltip.Content>
    </Tooltip.Root>
  </ToggleButtonGroup.Root>
);
```
