# ToggleButtonGroup

To group multiple `Button` or `IconButton` components together, visually and
logically, representing a set of related actions.

Available sizes.

```jsx-live
const App = () => {

  const sizes = ['2xl', 'xl', 'lg', 'md', 'sm', 'xs', '2xs'];
```
