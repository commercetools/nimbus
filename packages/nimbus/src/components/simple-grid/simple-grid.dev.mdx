## Getting started

### Import

```tsx
import { SimpleGrid, type SimpleGridProps } from '@commercetools/nimbus';
```

### Basic usage

SimpleGrid creates a grid layout with equal-sized children. Use it for quick, responsive layouts without complex grid configuration:

```jsx live-dev
const App = () => (
  <SimpleGrid columns={2} gap="400">
    <Box p="600" bg="info.7" borderRadius="200">
      <Text>Item 1</Text>
    </Box>
    <Box p="600" bg="info.7" borderRadius="200">
      <Text>Item 2</Text>
    </Box>
    <Box p="600" bg="info.7" borderRadius="200">
      <Text>Item 3</Text>
    </Box>
    <Box p="600" bg="info.7" borderRadius="200">
      <Text>Item 4</Text>
    </Box>
  </SimpleGrid>
)
```

## Component requirements

### Usage context

SimpleGrid should be used when you need:
- Quick, responsive grid layouts with minimal configuration
- Equal-sized grid items (product cards, image galleries, data cards)
- Auto-responsive layouts that adapt to container width
- Simple column-based layouts without complex positioning

**When to use SimpleGrid vs Grid:**

Use **SimpleGrid** when:
- All items should be equal size
- You need auto-responsive behavior with `minChildWidth`
- Layout is based on a simple column count
- You want minimal configuration

Use **Grid** when:
- Items need different sizes (row/column spanning)
- You need grid template areas for complex layouts
- Precise control over grid structure is required
- Creating dashboard layouts or complex page structures

## Accessibility

SimpleGrid is a layout component that doesn't add semantic meaning by itself. Ensure accessible markup:

#### Semantic Structure

- Wrap related content in appropriate semantic elements (`<article>`, `<section>`, `<nav>`)
- Use proper heading hierarchy within grid items
- Ensure grid items have meaningful content structure

```tsx
<SimpleGrid columns={3} gap="400" as="section" aria-labelledby="products-heading">
  <h2 id="products-heading" className="sr-only">Available Products</h2>
  {products.map(product => (
    <article key={product.id}>
      <h3>{product.name}</h3>
      <p>{product.description}</p>
    </article>
  ))}
</SimpleGrid>
```

#### Keyboard navigation

- Grid layout doesn't affect keyboard navigation
- Ensure interactive elements within grid items are keyboard accessible
- Maintain logical tab order

#### Persistent ID

If your use case requires tracking and analytics for this component, it is good practice to add a **persistent**, **unique** id to the component:

```tsx
const PERSISTENT_ID = "product-grid";

export const ProductGrid = () => (
  <SimpleGrid id={PERSISTENT_ID} columns={3} gap="400">
    <Box>Product 1</Box>
    <Box>Product 2</Box>
    <Box>Product 3</Box>
  </SimpleGrid>
);
```

## Common patterns

### Product grid

Display products in a responsive grid layout:

```jsx live-dev
const App = () => {
  const products = [
    { id: 1, name: "Wireless Mouse", price: "$29.99", stock: "In Stock" },
    { id: 2, name: "Mechanical Keyboard", price: "$89.99", stock: "In Stock" },
    { id: 3, name: "USB-C Hub", price: "$49.99", stock: "Low Stock" },
    { id: 4, name: "Monitor Stand", price: "$39.99", stock: "In Stock" },
    { id: 5, name: "Laptop Sleeve", price: "$24.99", stock: "In Stock" },
    { id: 6, name: "Cable Organizer", price: "$14.99", stock: "In Stock" },
  ];

  return (
    <SimpleGrid 
      columns={{ base: 1, sm: 2, md: 3 }}
      gap="400"
    >
      {products.map((product) => (
        <Box 
          key={product.id}
          p="600"
          bg="neutral.2"
          borderRadius="200"
          borderWidth="1px"
          borderColor="neutral.6"
          _hover={{ borderColor: "primary.7", bg: "neutral.3" }}
          transition="all 0.2s"
        >
          <Stack direction="column" gap="300">
            <Text fontWeight="600" fontSize="lg">{product.name}</Text>
            <Stack direction="row" justify="space-between" align="center">
              <Text fontWeight="600" color="primary.11">{product.price}</Text>
              <Text fontSize="sm" color={product.stock === "In Stock" ? "positive.11" : "warning.11"}>
                {product.stock}
              </Text>
            </Stack>
            <Button size="sm" variant="outline">Add to Cart</Button>
          </Stack>
        </Box>
      ))}
    </SimpleGrid>
  );
}
```

### Image gallery

Create responsive image galleries:

```jsx live-dev
const App = () => {
  const images = [
    { id: 1, alt: "Landscape 1", color: "primary.7" },
    { id: 2, alt: "Landscape 2", color: "info.7" },
    { id: 3, alt: "Landscape 3", color: "positive.7" },
    { id: 4, alt: "Landscape 4", color: "warning.7" },
    { id: 5, alt: "Landscape 5", color: "critical.7" },
    { id: 6, alt: "Landscape 6", color: "neutral.7" },
  ];

  return (
    <SimpleGrid minChildWidth="150px" gap="300">
      {images.map((image) => (
        <Box
          key={image.id}
          aspectRatio="4/3"
          bg={image.color}
          borderRadius="200"
          display="flex"
          alignItems="center"
          justifyContent="center"
          cursor="pointer"
          transition="transform 0.2s"
          _hover={{ transform: "scale(1.05)" }}
        >
          <Text color="white" fontWeight="600">{image.alt}</Text>
        </Box>
      ))}
    </SimpleGrid>
  );
}
```

### Dashboard widgets

Layout dashboard cards in a responsive grid:

```jsx live-dev
const App = () => {
  const stats = [
    { label: "Total Sales", value: "$12,345", change: "+12.5%", trend: "up" },
    { label: "Orders", value: "1,234", change: "+8.2%", trend: "up" },
    { label: "Customers", value: "5,678", change: "-2.4%", trend: "down" },
    { label: "Revenue", value: "$45,678", change: "+15.3%", trend: "up" },
  ];

  return (
    <SimpleGrid 
      columns={{ base: 1, sm: 2, lg: 4 }}
      gap="400"
    >
      {stats.map((stat, index) => (
        <Box
          key={index}
          p="600"
          bg="neutral.2"
          borderRadius="200"
          borderWidth="1px"
          borderColor="neutral.6"
        >
          <Stack direction="column" gap="200">
            <Text fontSize="sm" color="neutral.11">{stat.label}</Text>
            <Text fontSize="2xl" fontWeight="700">{stat.value}</Text>
            <Text 
              fontSize="sm" 
              fontWeight="600"
              color={stat.trend === "up" ? "positive.11" : "critical.11"}
            >
              {stat.change}
            </Text>
          </Stack>
        </Box>
      ))}
    </SimpleGrid>
  );
}
```
