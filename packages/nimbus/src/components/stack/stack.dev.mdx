# Stack Component

Stack is a layout component that manages the distribution and spacing of child elements in a single direction (horizontal or vertical). It's built on flexbox and provides a simple, declarative API for common layout patterns.

## Getting started

### Import

```tsx
import { Stack, type StackProps } from "@commercetools/nimbus";
```

### Basic usage

By default, Stack arranges children vertically with no gap:

```jsx live-dev
const App = () => (
  <Stack>
    <Box p="400" bg="primary.7" borderRadius="200">
      Item 1
    </Box>
    <Box p="400" bg="primary.7" borderRadius="200">
      Item 2
    </Box>
    <Box p="400" bg="primary.7" borderRadius="200">
      Item 3
    </Box>
  </Stack>
)
```

## Usage examples

### Direction variants

Control the layout direction with the `direction` prop. Supports `column` (default), `row`, `column-reverse`, and `row-reverse`:

```jsx live-dev
const App = () => (
  <Stack direction="column" gap="400">
    <Stack direction="row" gap="200">
      <Box p="400" bg="primary.7" borderRadius="200" flex="1">Row Item 1</Box>
      <Box p="400" bg="primary.7" borderRadius="200" flex="1">Row Item 2</Box>
      <Box p="400" bg="primary.7" borderRadius="200" flex="1">Row Item 3</Box>
    </Stack>
    
    <Stack direction="column" gap="200">
      <Box p="400" bg="info.7" borderRadius="200">Column Item 1</Box>
      <Box p="400" bg="info.7" borderRadius="200">Column Item 2</Box>
      <Box p="400" bg="info.7" borderRadius="200">Column Item 3</Box>
    </Stack>
    
    <Stack direction="row-reverse" gap="200">
      <Box p="400" bg="positive.7" borderRadius="200">Reverse 1</Box>
      <Box p="400" bg="positive.7" borderRadius="200">Reverse 2</Box>
      <Box p="400" bg="positive.7" borderRadius="200">Reverse 3</Box>
    </Stack>
  </Stack>
)
```

### Gap and spacing

Use the `gap` prop to control spacing between items. Accepts Nimbus spacing tokens:

```jsx live-dev
const App = () => {
  const [gap, setGap] = useState<StackProps["gap"]>("200");
  
  return (
    <Stack direction="column" gap="400">
      <Stack direction="row" gap="200" align="center">
        <Text fontWeight="600">Gap:</Text>
        <Button size="2xs" onClick={() => setGap("100")}>Small (100)</Button>
        <Button size="2xs" onClick={() => setGap("300")}>Medium (300)</Button>
        <Button size="2xs" onClick={() => setGap("600")}>Large (600)</Button>
      </Stack>
      
      <Stack direction="row" gap={gap}>
        <Box p="400" bg="primary.7" borderRadius="200">Item 1</Box>
        <Box p="400" bg="primary.7" borderRadius="200">Item 2</Box>
        <Box p="400" bg="primary.7" borderRadius="200">Item 3</Box>
      </Stack>
      
      <Text fontSize="sm" color="neutral.11">
        Current gap: {gap}
      </Text>
    </Stack>
  );
}
```

```jsx live-dev
        </Stack>
        <Stack direction="column" gap="200" align="flex-end" bg="neutral.3" p="400" borderRadius="200" flex="1">

      </Stack>
    </Box>
  </Stack>
)
```

### Wrapping behavior

Control how items wrap with the `wrap` prop:

```jsx live-dev
const App = () => (
  <Stack direction="column" gap="400">
    <Box>
      <Text fontWeight="600" mb="200">Wrap (default):</Text>
      <Stack direction="row" wrap="wrap" gap="200" bg="neutral.3" p="400" borderRadius="200">
        {Array.from({ length: 12 }, (_, i) => (
          <Box key={i} p="300" bg="primary.7" borderRadius="200" minWidth="1600">
            Item {i + 1}
          </Box>
        ))}
      </Stack>
    </Box>
    
    <Box>
      <Text fontWeight="600" mb="200">No wrap (overflow):</Text>
      <Stack direction="row" wrap="nowrap" gap="200" bg="neutral.3" p="400" borderRadius="200" overflow="auto">
        {Array.from({ length: 12 }, (_, i) => (
          <Box key={i} p="300" bg="info.7" borderRadius="200" minWidth="1600">
            Item {i + 1}
          </Box>
        ))}
      </Stack>
    </Box>
  </Stack>
)
```

### Responsive layouts

Stack supports responsive prop values for different breakpoints:

```jsx live-dev
const App = () => (
  <Stack 
    direction={{ base: "column", md: "row" }}
    gap={{ base: "300", md: "600" }}
    align={{ base: "stretch", md: "center" }}
  >
    <Box p="600" bg="primary.7" borderRadius="200" flex="1">
      <Text fontWeight="600" mb="200">Card 1</Text>
      <Text fontSize="sm">Stacks vertically on mobile, horizontally on desktop</Text>
    </Box>
    <Box p="600" bg="primary.7" borderRadius="200" flex="1">
      <Text fontWeight="600" mb="200">Card 2</Text>
      <Text fontSize="sm">Resize the window to see the layout change</Text>
    </Box>
    <Box p="600" bg="primary.7" borderRadius="200" flex="1">
      <Text fontWeight="600" mb="200">Card 3</Text>
      <Text fontSize="sm">Gap also adjusts responsively</Text>
    </Box>
  </Stack>
)
```

## Component requirements

### Usage context

Stack should be used when you need to:
- Arrange multiple elements in a single direction (horizontal or vertical)
- Apply consistent spacing between elements
- Add separators between items
- Create responsive layouts that change direction based on viewport

For more complex grid-based layouts, consider using the `Grid` component. For simple flex containers with more control over individual item properties, consider using `Flex` or `Box`.

## Accessibility

Stack is a layout component with no inherent accessibility requirements. However:

- Ensure child elements have appropriate accessibility attributes
- Use semantic HTML elements within Stack when appropriate
- Consider reading order when using `row-reverse` or `column-reverse`
- Test keyboard navigation flows through stacked elements

#### Persistent ID

If your use case requires tracking and analytics for this component, it is good practice to add a **persistent**, **unique** id to the component:

```tsx
<Stack id="main-navigation-stack">
  {/* children */}
</Stack>
```

## Common patterns

### Navigation bar with actions

Use Stack to create navigation layouts with items on opposite ends:

```jsx live-dev
const App = () => (
  <Stack direction="row" align="center" gap="400" p="400" bg="neutral.2" borderRadius="200">
    <Text fontWeight="600" fontSize="lg">Logo</Text>
    <Spacer />
    <Stack direction="row" gap="200">
      <Button size="xs" variant="ghost">Home</Button>
      <Button size="xs" variant="ghost">Products</Button>
      <Button size="xs" variant="ghost">About</Button>
      <Button size="xs" variant="outline">Sign In</Button>
    </Stack>
  </Stack>
)
```
