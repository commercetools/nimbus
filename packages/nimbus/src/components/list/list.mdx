# List

The list component organizes related items or data into a vertical, scannable
format.

## Overview

The list component presents structured content which can include details such as
text, icons, and actions, in a vertical arrangement. With improved readability
letting readers quickly scan the content and compare relevant details.

### Basic options

#### Vertical lists

Vertical lists can function differently based on need. Using bullet points,
numbered lists, and icons or emojis will help users understand the information
differently as needed.

#### Alignment

List items are aligned to the left, center and the right. Choose what suits the
information to be shown based on language and data.

#### Nested items, added gap space, and item wrapping

When nesting related items, maintain proper spacing and wrapping to keep the
structure intact. Avoid long lines of text; if necessary, rephrase to make it
shorter or use body text instead of bullet points, which should be reserved for
concise information.

```jsx live
const App = () => (
  <Stack direction="horizontal" gap="800" alignItems="flex-start">
    <Stack gap="200">
      <Text fontWeight="bold">Nested items</Text>
      <List.Root>
        <List.Item>
          Password requirements:
          <List.Root pl="400">
            <List.Item>8 characters and 3 numbers</List.Item>
            <List.Item>One special character (!@#$%^&*)</List.Item>
          </List.Root>
        </List.Item>
        <List.Item>
          Name and email requirements
          <List.Root pl="400">

          </List.Root>
        </List.Item>
      </List.Root>
    </Stack>
    <Stack gap="200">
      <Text fontWeight="bold">Gap space</Text>
      <List.Root gap="400">

          Password requirements:
```
