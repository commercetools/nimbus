## Overview

A progress bar's primary usage is to visually communicate to the user that a
process is ongoing. If used in combination with a flow or tasks, a progress bar
can give a visual indicator of where they currently are in the process and
estimate how much is left to complete the task.

## Variables

Get familiar with the features.

### Visuals

#### On white background

Most common, progress bar is used on white backgrounds.

```jsx live
const App = () => (
  <ProgressBar
    value={50}
    label=""
    colorPalette="primary"
  />
)
```

#### On dark background

If being used on a dark background, use the contrast variant.

```jsx live
const App = () => (
  <Box bg="neutral.9" p="600">
    <ProgressBar
      value={50}
      label=""
      variant="contrast"
      colorPalette="primary"
    />
  </Box>
)
```

### Size

There are two sizes to progress bar: the default size is medium (md) and
2x-small (2xs).

```jsx live
const App = () => (
  <Stack direction="column" gap="50" alignItems="stretch">
    <ProgressBar
      value={75}
      label=""
      size="2xs"
      colorPalette="primary"
    />
    <ProgressBar
      value={75}
      label=""
      size="md"
      colorPalette="primary"
    />
  </Stack>
)
```

### Layouts

Customizable layouts for different use cases, and to fit the UI best.

```jsx live
const App = () => (
  <Stack direction="column" gap="200" alignItems="stretch">
    <ProgressBar
      value={50}
      label="Stacked label"
      layout="stacked"
      colorPalette="primary"
    />
    <ProgressBar
      value={50}
      label="Inline label"
      layout="inline"
      colorPalette="primary"
    />
  </Stack>
)
```

### Animation logic

Two options for loading animations based on timing.

#### Dynamic

A determinate progress bar indicates a specific percentage of completion. This
is useful when the total time or size of a task is known. For dynamic loading,
the color gradient is animated.

```jsx live
const App = () => {
  const [value, setValue] = React.useState(0);
  const [isRunning, setIsRunning] = React.useState(false);

  React.useEffect(() => {
    if (!isRunning) return;

    const interval = setInterval(() => {
      setValue((prev) => {
        if (prev >= 100) {
          setIsRunning(false);
          return 100;
        }
        return prev + 10;
      });
    }, 500);

    return () => clearInterval(interval);
  }, [isRunning]);

  const handleStart = () => {
    setValue(0);
    setIsRunning(true);
  };

  return (
    <Stack direction="column" gap="400" alignItems="stretch">
      <ProgressBar
        value={value}
        label="Loading..."
        colorPalette="primary"
      />
      <Button
        onClick={handleStart}
        isDisabled={isRunning}
        width="fit-content"
        colorPalette="primary"
        variant="solid"
      >
        Start
      </Button>
    </Stack>
  )
}
```

#### Static

A static loading bar allows a viewer to know where they are currently in the
process. Static progress bars do not have an animated color gradient.

```jsx live
const App = () => {
  const [currentLesson, setCurrentLesson] = React.useState(5);
  const totalLessons = 10;
  const progress = (currentLesson / totalLessons) * 100;

  const handleNextLesson = () => {
    if (currentLesson < totalLessons) {
      setCurrentLesson(prev => prev + 1);
    }
  };

  return (
    <Stack direction="column" gap="400" alignItems="stretch">
      <ProgressBar
        value={progress}
        label="Lessons completed"
        colorPalette="primary"
      />
      <Button
        variant="outline"
        width="fit-content"
        onClick={handleNextLesson}
        isDisabled={currentLesson >= totalLessons}
      >
        Next lesson
      </Button>
    </Stack>
  )
}
```

#### Indeterminate

An indeterminate progress bar shows that an operation is in progress without
indicating how much of it is complete. This is useful for tasks with an unknown
duration.

```jsx live
const App = () => (
  <ProgressBar
    isIndeterminate={true}
    label="Uploading your files..."
    colorPalette="primary"
  />
)
```
