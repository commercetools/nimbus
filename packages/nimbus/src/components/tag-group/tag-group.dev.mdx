## Getting started

### Import

```tsx
import { TagGroup } from '@commercetools/nimbus';
```

### Basic usage

The TagGroup component is a compound component that displays a collection of tags. It consists of three parts: `TagGroup.Root`, `TagGroup.TagList`, and `TagGroup.Tag`:

```jsx-live-dev
const App = () => (
  <TagGroup.Root aria-label="Categories">
    <TagGroup.TagList>
      <TagGroup.Tag>Electronics</TagGroup.Tag>
      <TagGroup.Tag>Clothing</TagGroup.Tag>
      <TagGroup.Tag>Home & Garden</TagGroup.Tag>
    </TagGroup.TagList>
  </TagGroup.Root>
)
```

### Dynamic collections with items

Use the `items` prop with a render function for dynamic data:

```jsx-live-dev
const App = () => {
  const categories = [
    { id: 'electronics', name: 'Electronics' },
    { id: 'clothing', name: 'Clothing' },
    { id: 'home', name: 'Home & Garden' },
    { id: 'sports', name: 'Sports & Outdoors' },
  ];

  return (
    <TagGroup.Root aria-label="Product categories">
      <TagGroup.TagList items={categories}>
        {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
      </TagGroup.TagList>
    </TagGroup.Root>
  );
}
```

The `items` prop accepts an array of objects, and the render function receives each item as an argument. This pattern is useful when working with dynamic data from APIs or state management.

### Color palette

Customize tag colors by setting the `colorPalette` prop on individual `TagGroup.Tag` components. This prop accepts semantic colors (primary, info, critical, warning, positive) or system colors (blue, green, red, purple, etc.) to establish the color context for that tag.

Note that the `colorPalette` prop selects which palette to use, but doesn't immediately change the visuals. You must also reference color tokens (e.g. `backgroundColor="colorPalette.11"`, `color="colorPalette.6"`) to actually apply the intended colors.

### Removable tags

Enable tag removal by providing an `onRemove` handler. Users can remove tags by clicking the remove button or pressing Backspace:

```jsx-live-dev
const App = () => {
  const [tags, setTags] = useState([
    { id: 'react', name: 'React' },
    { id: 'typescript', name: 'TypeScript' },
    { id: 'nodejs', name: 'Node.js' },
    { id: 'graphql', name: 'GraphQL' },
  ]);

  const handleRemove = (keys) => {
    setTags(tags.filter(tag => !keys.has(tag.id)));
  };

  return (
    <Stack direction="column" gap="300">
      <TagGroup.Root 
        aria-label="Tech stack" 
        onRemove={handleRemove}
      >
        <TagGroup.TagList items={tags}>
          {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
        </TagGroup.TagList>
      </TagGroup.Root>
      <Text fontSize="sm" color="fg.muted">
        {tags.length} {tags.length === 1 ? 'tag' : 'tags'} remaining
      </Text>
    </Stack>
  );
}
```

**Keyboard interaction**: When focused on a tag, press `Backspace` or `Delete` to remove it.

### Single selection

Use `selectionMode="single"` to allow selecting one tag at a time:

```jsx-live-dev
const App = () => {
  const [selected, setSelected] = useState(new Set());

  const priorities = [
    { id: 'low', name: 'Low' },
    { id: 'medium', name: 'Medium' },
    { id: 'high', name: 'High' },
    { id: 'urgent', name: 'Urgent' },
  ];

  return (
    <Stack direction="column" gap="300">
      <TagGroup.Root
        aria-label="Select priority"
        selectionMode="single"
        selectedKeys={selected}
        onSelectionChange={setSelected}
      >
        <TagGroup.TagList items={priorities}>
          {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
        </TagGroup.TagList>
      </TagGroup.Root>
      <Text fontSize="sm">
        Selected: {selected.size > 0 ? [...selected][0] : 'None'}
      </Text>
    </Stack>
  );
}
```

### Multiple selection

Use `selectionMode="multiple"` to allow selecting multiple tags:

```jsx-live-dev
const App = () => {
  const [selected, setSelected] = useState(new Set(['electronics']));

  const filters = [
    { id: 'electronics', name: 'Electronics' },
    { id: 'clothing', name: 'Clothing' },
    { id: 'home', name: 'Home & Garden' },
    { id: 'sports', name: 'Sports' },
    { id: 'books', name: 'Books' },
  ];

  return (
    <Stack direction="column" gap="300">
      <TagGroup.Root
        aria-label="Filter categories"
        selectionMode="multiple"
        selectedKeys={selected}
        onSelectionChange={setSelected}
      >
        <TagGroup.TagList items={filters}>
          {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
        </TagGroup.TagList>
      </TagGroup.Root>
      <Text fontSize="sm">
        {selected.size} {selected.size === 1 ? 'filter' : 'filters'} selected
      </Text>
    </Stack>
  );
}
```

**Note**: When using `selectionMode="single"` or `selectionMode="multiple"`, tags cannot be removed. These modes are mutually exclusive with the removal functionality.

## Component requirements

### Accessibility

The TagGroup component is built on React Aria's TagGroup, which provides comprehensive accessibility support. However, you must always provide an accessible label:

**Provide a label** using one of these methods:

- Using a visible label with `aria-labelledby`:

```tsx
<Text id="tag-label">Categories</Text>
<TagGroup.Root aria-labelledby="tag-label">
  <TagGroup.TagList>
    <TagGroup.Tag>Electronics</TagGroup.Tag>
  </TagGroup.TagList>
</TagGroup.Root>
```

- Using `aria-label` for a non-visible label:

```tsx
<TagGroup.Root aria-label="Product categories">
  <TagGroup.TagList>
    <TagGroup.Tag>Electronics</TagGroup.Tag>
  </TagGroup.TagList>
</TagGroup.Root>
```

If your use case requires tracking and analytics for this component, it is good practice to add a **persistent**, **unique** id to the component:

```tsx
const PERSISTENT_ID = "example-tag-group";

export const Example = () => (
  <TagGroup.Root id={PERSISTENT_ID} aria-label="Categories">
    <TagGroup.TagList>
      <TagGroup.Tag>Electronics</TagGroup.Tag>
    </TagGroup.TagList>
  </TagGroup.Root>
);
```

#### Keyboard navigation

The component supports full keyboard interaction:

- `Tab`: Moves focus into and out of the tag group
- `Arrow Left` / `Arrow Right`: Navigates between tags within the group
- `Enter` / `Space`: Selects the focused tag (when selection mode is enabled)
- `Backspace` / `Delete`: Removes the focused tag (when `onRemove` is provided)
- Focus wraps from the last tag to the first and vice versa

## Common patterns

### Filter chips with removal

Display active filters that can be removed:

```jsx-live-dev
const App = () => {
  const [filters, setFilters] = useState([
    { id: 'price-low', name: 'Price: Low to High', category: 'sort' },
    { id: 'instock', name: 'In Stock', category: 'availability' },
    { id: 'brand-apple', name: 'Apple', category: 'brand' },
  ]);

  const handleRemove = (keys) => {
    setFilters(filters.filter(f => !keys.has(f.id)));
  };

  const handleClearAll = () => {
    setFilters([]);
  };

  return (
    <Stack direction="column" gap="300">
      <Stack direction="row" justify="space-between" align="center">
        <Text fontSize="sm" fontWeight="semibold">Active Filters</Text>
        {filters.length > 0 && (
          <Button size="sm" variant="ghost" onPress={handleClearAll}>
            Clear all
          </Button>
        )}
      </Stack>
      {filters.length > 0 ? (
        <TagGroup.Root 
          aria-label="Active filters" 
          onRemove={handleRemove}
          colorPalette="blue"
        >
          <TagGroup.TagList items={filters}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      ) : (
        <Text fontSize="sm" color="fg.muted">No filters applied</Text>
      )}
    </Stack>
  );
}
```

### Category selector

Use single selection for choosing one category:

```jsx-live-dev
const App = () => {
  const [selected, setSelected] = useState(new Set(['all']));
  const [itemCount, setItemCount] = useState(124);

  const categories = [
    { id: 'all', name: 'All Products', count: 124 },
    { id: 'new', name: 'New Arrivals', count: 18 },
    { id: 'sale', name: 'On Sale', count: 32 },
    { id: 'featured', name: 'Featured', count: 12 },
  ];

  const handleSelectionChange = (keys) => {
    setSelected(keys);
    const selectedId = [...keys][0];
    const category = categories.find(c => c.id === selectedId);
    setItemCount(category?.count || 0);
  };

  return (
    <Stack direction="column" gap="300">
      <TagGroup.Root
        aria-label="Product categories"
        selectionMode="single"
        selectedKeys={selected}
        onSelectionChange={handleSelectionChange}
      >
        <TagGroup.TagList items={categories}>
          {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
        </TagGroup.TagList>
      </TagGroup.Root>
      <Text fontSize="sm" color="fg.muted">
        Showing {itemCount} products
      </Text>
    </Stack>
  );
}
```

## Testing your implementation

These examples demonstrate how to test your implementation when using TagGroup in your application. The component's internal functionality is already tested by Nimbus - these patterns help you verify your integration and application-specific logic.

### Basic rendering tests

Verify the component renders with expected tags:

```tsx
import { render, screen } from '@testing-library/react';
import { TagGroup } from '@commercetools/nimbus';

describe('TagGroup', () => {
  it('renders tags with content', () => {
    render(
      <TagGroup.Root aria-label="Categories">
        <TagGroup.TagList>
          <TagGroup.Tag>Electronics</TagGroup.Tag>
          <TagGroup.Tag>Clothing</TagGroup.Tag>
        </TagGroup.TagList>
      </TagGroup.Root>
    );

    expect(screen.getByText('Electronics')).toBeInTheDocument();
    expect(screen.getByText('Clothing')).toBeInTheDocument();
  });

  it('renders with dynamic items', () => {
    const items = [
      { id: 'electronics', name: 'Electronics' },
      { id: 'clothing', name: 'Clothing' },
    ];

    render(
      <TagGroup.Root aria-label="Categories">
        <TagGroup.TagList items={items}>
          {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
        </TagGroup.TagList>
      </TagGroup.Root>
    );

    expect(screen.getByText('Electronics')).toBeInTheDocument();
    expect(screen.getByText('Clothing')).toBeInTheDocument();
  });
});
```

### Selection tests

Test single and multiple selection modes:

```tsx
import { render, screen, waitFor } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import { TagGroup } from '@commercetools/nimbus';

describe('TagGroup selection', () => {
  it('handles single selection', async () => {
    const user = userEvent.setup();
    const handleSelectionChange = jest.fn();

    render(
      <TagGroup.Root
        aria-label="Priority"
        selectionMode="single"
        onSelectionChange={handleSelectionChange}
      >
        <TagGroup.TagList>
          <TagGroup.Tag id="low">Low</TagGroup.Tag>
          <TagGroup.Tag id="high">High</TagGroup.Tag>
        </TagGroup.TagList>
      </TagGroup.Root>
    );

    const lowTag = screen.getByText('Low');
    await user.click(lowTag);

    await waitFor(() => {
      expect(handleSelectionChange).toHaveBeenCalled();
      const selection = handleSelectionChange.mock.calls[0][0];
      expect(selection.has('low')).toBe(true);
    });
  });

  it('handles multiple selection', async () => {
    const user = userEvent.setup();
    const handleSelectionChange = jest.fn();

    render(
      <TagGroup.Root
        aria-label="Filters"
        selectionMode="multiple"
        onSelectionChange={handleSelectionChange}
      >
        <TagGroup.TagList>
          <TagGroup.Tag id="tag1">Tag 1</TagGroup.Tag>
          <TagGroup.Tag id="tag2">Tag 2</TagGroup.Tag>
        </TagGroup.TagList>
      </TagGroup.Root>
    );

    await user.click(screen.getByText('Tag 1'));
    await user.click(screen.getByText('Tag 2'));

    await waitFor(() => {
      expect(handleSelectionChange).toHaveBeenCalledTimes(2);
    });
  });
});
```

### Removal tests

Test tag removal functionality:

```tsx
import { render, screen, waitFor } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import { TagGroup } from '@commercetools/nimbus';

describe('TagGroup removal', () => {
  it('calls onRemove when remove button is clicked', async () => {
    const user = userEvent.setup();
    const handleRemove = jest.fn();

    render(
      <TagGroup.Root aria-label="Tags" onRemove={handleRemove}>

    const removeButton = screen.getByLabelText('Remove tag');

      expect(handleRemove).toHaveBeenCalledWith(
        expect.objectContaining({
          has: expect.any(Function),
        })
      );
    });
  });

  it('removes tag with keyboard', async () => {
    const user = userEvent.setup();
    const handleRemove = jest.fn();

          <TagGroup.Tag id="tag1">Tag 1</TagGroup.Tag>
```

### Keyboard navigation tests

Test keyboard interactions:

```tsx
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import { TagGroup } from '@commercetools/nimbus';

describe('TagGroup keyboard navigation', () => {
  it('navigates between tags with arrow keys', async () => {
    const user = userEvent.setup();

    render(
      <TagGroup.Root aria-label="Tags">
        <TagGroup.TagList>
          <TagGroup.Tag>First</TagGroup.Tag>
          <TagGroup.Tag>Second</TagGroup.Tag>
          <TagGroup.Tag>Third</TagGroup.Tag>
        </TagGroup.TagList>
      </TagGroup.Root>
    );

    // Tab to enter the tag group
    await user.tab();
    expect(document.activeElement?.textContent).toContain('First');

    // Arrow right to next tag
    await user.keyboard('{ArrowRight}');
    expect(document.activeElement?.textContent).toContain('Second');

    // Arrow left to previous tag
    await user.keyboard('{ArrowLeft}');
    expect(document.activeElement?.textContent).toContain('First');
  });
});
```
