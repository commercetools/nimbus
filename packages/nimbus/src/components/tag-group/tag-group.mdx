# Tag group

A tag group is a compact, interactive element. It categorizes, labels, or adds
attributes to objects for quick recognition and filtering.

## Overview

The tag group organizes a collection of individual tag components, allowing
users to efficiently scan and understand multiple pieces of metadata or filter
criteria associated with a product, order, or entity in a compact, manageable
space.

### Visual options

#### Size

The large size of the tag within tag groups is the default, the small size is
meant for more compact uses. The more tags present, consider smaller sizes to
make the information more compact and easily read for the user.

```jsx live
const App = () => {
  const smallTag = [{ id: "cherries", name: "Cherries" }];
  const mediumTag = [{ id: "kiwis", name: "Kiwis" }];
  const largeTag = [{ id: "apples", name: "Apples" }];

  return (
    <Stack direction="row" gap="400" alignItems="flex-start">
      <Box>
        <Text as="p" textStyle="sm" fontWeight="bold" marginBottom="200">Large</Text>
        <TagGroup.Root size="lg" aria-label="Large tags" onRemove={() => {}}>
          <TagGroup.TagList items={largeTag}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      </Box>
      <Box>
        <Text as="p" textStyle="sm" fontWeight="bold" marginBottom="200">Medium</Text>
        <TagGroup.Root size="md" aria-label="Medium tags" onRemove={() => {}}>
          <TagGroup.TagList items={mediumTag}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      </Box>
      <Box>
        <Text as="p" textStyle="sm" fontWeight="bold" marginBottom="200">Small</Text>
        <TagGroup.Root size="sm" aria-label="Small tags" onRemove={() => {}}>
          <TagGroup.TagList items={smallTag}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      </Box>
    </Stack>
  );
}
```

### Toggle buttons vs clearable tag groups

Allow users to toggle the active state of a tag, or be able to remove tags by
adding a "close" icon to each tag. Avoid mixing these interactions within the
same group.

```jsx live
const App = () => {
  const [selectedToggleTags, setSelectedToggleTags] = useState(new Set(["vegetables"]));

  const removableTags = [
    { id: "remove1", name: "Remove me first" },
    { id: "remove2", name: "Remove me second" },
    { id: "remove3", name: "Remove me last" },
  ];

  const toggleTags = [
    { id: "fruits", name: "Fruits" },
    { id: "vegetables", name: "Vegetables" },
    { id: "meat", name: "Meat" },
  ];

  return (
    <Stack direction="column" gap="600" alignItems="flex-start">
      <Box>
        <Text as="p" textStyle="sm" marginBottom="200" fontWeight="bold">Removable tag group</Text>
        <TagGroup.Root aria-label="Removable tags" onRemove={() => {}}>
          <TagGroup.TagList items={removableTags}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      </Box>

      <Box>
        <Text as="p" textStyle="sm" marginBottom="200" fontWeight="bold">Toggleable tag group</Text>
        <TagGroup.Root
          aria-label="Toggle tags"
          selectionMode="multiple"
          selectedKeys={selectedToggleTags}
          onSelectionChange={setSelectedToggleTags}
        >
          <TagGroup.TagList items={toggleTags}>
            {(item) => <TagGroup.Tag>{item.name}</TagGroup.Tag>}
          </TagGroup.TagList>
        </TagGroup.Root>
      </Box>
    </Stack>
  );
}
```
