# Group

Groups related elements together for better organization and layout.

## Overview

The Group component provides a flexible way to group related UI elements with consistent spacing and alignment. It's particularly useful for creating forms, toolbars, and other collections of interactive elements.

## Variables

Get familiar with the features.

### Basic Usage

```jsx-live
const App = () => (
  <Group>
    <Button>First</Button>
    <Button>Second</Button>
    <Button>Third</Button>
  </Group>
)
```
