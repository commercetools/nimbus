## Overview

Composed of several elements, a label, optional helper icon, a text input. There
is also an optional helper text feature that presents at the bottom of the
field.

### Resources

Deep dive into implementation details and access the Nimbus design library.

[Figma library](https://www.figma.com/design/AvtPX6g7OGGCRvNlatGOIY/NIMBUS-design-system?node-id=349-2635&m=dev)

## Variables

Get familiar with the features.

### Label options

#### Simple

Label is used to describe the form.

```jsx live
const App = () => {
  const [val, setVal] = useState('Input value');

  return (
    <FormField.Root>
      <FormField.Label>Label</FormField.Label>
      <FormField.Input>
        <TextInput width="full" value={val} onChange={(v)=>setVal(v)} placeholder="Enter something..." />
      </FormField.Input>
    </FormField.Root>
  )
}
```

#### With help text

Label paired with form and description to give users extra clarification.

```jsx live
const App = () => {
  const [val, setVal] = useState('Input value');

  return (
    <FormField.Root>
      <FormField.Label>Label</FormField.Label>

      </FormField.Input>

    </FormField.Root>
  )
}
```

#### With error message

Explain errors to the user in an eye catching way.

```jsx live
const App = () => {
  const [val, setVal] = useState('Input value');

  return (
    <FormField.Root isInvalid>
      <FormField.Label>Label</FormField.Label>
      <FormField.Input>
        <TextInput width="full" value={val} onChange={(v)=>setVal(v)} placeholder="Enter something..." />
      </FormField.Input>
      <FormField.Error>
        Error message
      </FormField.Error>
    </FormField.Root>
  )
}
```

#### Required with info icon and help text

This is the default placement of labels. Info button should always open an info
dialog, not just a tooltip on hover.

```jsx live
const App = () => {
  const [val, setVal] = useState('Input value');

  return (
    <FormField.Root isRequired>
      <FormField.Label>Label</FormField.Label>
      <FormField.Input>
        <TextInput width="full" value={val} onChange={(v)=>setVal(v)} placeholder="Enter something..." />
      </FormField.Input>
      <FormField.Description>
        Help Text
      </FormField.Description>
      <FormField.InfoBox>
        Info box content.
      </FormField.InfoBox>
    </FormField.Root>
  )
}
```
