## Getting started

### Import

```tsx
import { LoadingSpinner, type LoadingSpinnerProps } from '@commercetools/nimbus';
```

### Basic usage

The LoadingSpinner component indicates an ongoing process or indeterminate loading state.

```jsx-live-dev
const App = () => (
  <LoadingSpinner />
)
```

## Usage examples

### Size options

The `size` prop controls the dimensions of the spinner to match your interface density. Available sizes are `2xs`, `xs`, `sm` (default), `md`, and `lg`.

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400" alignItems="center">
    <LoadingSpinner size="2xs" />
    <LoadingSpinner size="xs" />
    <LoadingSpinner size="sm" />
    <LoadingSpinner size="md" />
    <LoadingSpinner size="lg" />
  </Stack>
)
```

### Visual variants

The `colorPalette` prop enables rendering a high-contrast loading spinner on light and dark backgrounds.

- `primary`: The default color, suitable for light backgrounds.
- `white`: Suitable for dark backgrounds or colored surfaces.

```jsx-live-dev
const App = () => (
  <Stack direction="row" gap="400">
    <LoadingSpinner colorPalette="primary" margin="400"/>
    <LoadingSpinner colorPalette="white" margin="400" bgColor="black" />
  </Stack>
)
```
