## Getting started

### Import

```tsx
import { LoadingSpinner, type LoadingSpinnerProps } from '@commercetools/nimbus';
```

### Basic usage

The LoadingSpinner component indicates an ongoing process or indeterminate loading state.

```jsx live-dev
const App = () => (
  <LoadingSpinner />
)
```

## Usage examples

### Size options

The `size` prop controls the dimensions of the spinner to match your interface density. Available sizes are `2xs`, `xs`, `sm` (default), `md`, and `lg`.

```jsx live-dev
const App = () => (
  <Stack direction="row" gap="400" alignItems="center">
    <LoadingSpinner size="2xs" />
    <LoadingSpinner size="xs" />
    <LoadingSpinner size="sm" />
    <LoadingSpinner size="md" />
    <LoadingSpinner size="lg" />
  </Stack>
)
```

### Visual variants

The `colorPalette` prop enables rendering a high-contrast loading spinner on light and dark backgrounds.

- `primary`: The default color, suitable for light backgrounds.
- `white`: Suitable for dark backgrounds or colored surfaces.

```jsx live-dev
const App = () => (
  <Stack direction="row" gap="400">
    <LoadingSpinner colorPalette="primary" margin="400"/>
    <LoadingSpinner colorPalette="white" margin="400" bgColor="black" />
  </Stack>
)
```

## Component requirements

## Accessibility

The LoadingSpinner uses `role="progressbar"` and an indeterminate state by default.

You **must** ensure the spinner has an accessible label. The component provides a default localized label ("Loading"), but you should provide a more specific label when possible using `aria-label` or `aria-labelledby`.

```tsx
// Recommended: Specific label describing what is loading
<LoadingSpinner aria-label="Loading order details" />

// Using aria-labelledby
<h3 id="loading-heading">Loading Users</h3>
<LoadingSpinner aria-labelledby="loading-heading" />
```

If your use case requires tracking and analytics for this component, it is good practice to add a **persistent**, **unique** id to the component:

```tsx
const PERSISTENT_ID = "user-list-loading-spinner";

export const UserList = () => (
  <LoadingSpinner id={PERSISTENT_ID} />
);
```
