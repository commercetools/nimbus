# LoadingSpinner

The LoadingSpinner component provides visual feedback to users that an operation is in progress. It helps improve the user experience by indicating that the application is working on a task rather than being stuck or unresponsive.

## Basic Usage

In its simplest form, the LoadingSpinner can be used without any props to show the default loading indicator.

```jsx-live
const App = () => <LoadingSpinner />
```

## Sizes

The LoadingSpinner component comes in different sizes to accommodate various use cases. You can adjust the size using the `size` prop.

```jsx-live
const App = () => {
  const sizes = ['xs', 'sm', 'md', 'lg', 'xl'];

  return (
    <Stack direction="horizontal" alignItems="center" gap="md">
      {sizes.map(size => (
        <LoadingSpinner key={size} size={size} />
      ))}
    </Stack>
  )
}
```

## Tones

You can customize the color of the LoadingSpinner using the `tone` prop to match your application's design system.

```jsx-live
const App = () => {
  const tones = ['primary', 'white'];

  return (
    <Stack direction="horizontal" alignItems="center" gap="md">
    
      
      <Box p="400"><LoadingSpinner tone="primary" size="md" /></Box>
      <Box p="400" bg="neutral.12"><LoadingSpinner tone="white" size="md" /></Box>
    </Stack>
  )
}
```
