## Overview

A time input enables users to enter and select a specific time. It offers
flexible input methods while maintaining accuracy and usability.

## Variables

Get familiar with the features.

### Visual options

#### Size

Medium size is the default sizing used most commonly, with a small size for
compact use cases.

```jsx live
const App = () => (
  <Grid templateColumns="repeat(2, 1fr)" gap="400">
    <TimeInput
      size="md"
      hourCycle={24}
      defaultValue={new Time(20, 0)}
    />
    <TimeInput
      size="sm"
      hourCycle={12}
      defaultValue={new Time(9, 30)}
    />
  </Grid>
);
```

#### Solid appearance

This is the default styling, it provides a more prominent, filled style for
standard form usage where the input needs to stand out clearly.

```jsx live
const App = () => (
  <TimeInput
    size="md"
    hourCycle={12}
    defaultValue={new Time(15, 45)}
    width="50%"
  />
);
```

#### Ghost appearance

Offers a minimalist style (e.g., no background fill, just a border) for
integration into uncluttered interfaces or specific contexts.

```jsx live
const App = () => (
  <TimeInput
    size="md"
    variant="ghost"
    hourCycle={24}
    defaultValue={new Time(7, 0)}
    width="50%"
  />
);
```
