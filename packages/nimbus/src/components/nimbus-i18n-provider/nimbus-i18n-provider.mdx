## Overview

NimbusI18nProvider is a proxy component that wraps React Aria's I18nProvider,
making it easy to set the locale for all React Aria components like DatePicker,
Calendar, and NumberInput. This ensures consistent formatting across your
application.

### Key features

- **Locale-aware formatting**: Controls date, number, and currency formatting
  based on locale
- **React Aria integration**: Built specifically for React Aria components
- **BCP47 locale codes**: Supports standard language-region codes (e.g.,
  'en-US', 'de-DE')
- **Type-safe**: Full TypeScript support
- **Automatic propagation**: Locale settings cascade to all child React Aria
  components

[React Aria I18nProvider Docs](https://react-spectrum.adobe.com/react-aria/I18nProvider.html)

## Variables

Get familiar with the features.

### Basic usage

Wrap your application with NimbusI18nProvider to set the locale for React Aria
components:

```jsx live
const App = () => (
  <NimbusI18nProvider locale="en-US">
    <Stack direction="column" gap="400">
      <Text>This app uses US English formatting</Text>
      <DatePicker label="Select Date" />
    </Stack>
  </NimbusI18nProvider>
);
```

### Different locales

Change the locale prop to switch formatting:

```jsx live
const App = () => {
  const [locale, setLocale] = React.useState("en-US");

  return (
    <Stack direction="column" gap="400">
      <Stack direction="row" gap="200" wrap="wrap">
        <Button onClick={() => setLocale("en-US")} size="sm">
          English (US)
        </Button>
        <Button onClick={() => setLocale("en-GB")} size="sm">
          English (UK)
        </Button>
        <Button onClick={() => setLocale("de-DE")} size="sm">
          Deutsch
        </Button>
        <Button onClick={() => setLocale("es-ES")} size="sm">
          Español
        </Button>
        <Button onClick={() => setLocale("fr-FR")} size="sm">
          Français
        </Button>
        <Button onClick={() => setLocale("ja-JP")} size="sm">
          日本語
        </Button>
      </Stack>

      <NimbusI18nProvider locale={locale}>
        <Stack direction="column" gap="400">
          <Text fontWeight="600">Current Locale: {locale}</Text>
          <DatePicker label="Select Date" />
          <NumberInput
            label="Amount"
            defaultValue={1234.56}
            minValue={0}
            formatOptions={{
              style: "currency",
              currency: locale.startsWith("ja") ? "JPY" : locale.startsWith("en-GB") ? "GBP" : "USD"
            }}
          />
        </Stack>
      </NimbusI18nProvider>
    </Stack>
  );
};
```

### Multiple providers

You can nest NimbusI18nProviders to use different locales in different parts of
your app:

```jsx live
const App = () => (
  <Stack direction="column" gap="600">
    <NimbusI18nProvider locale="en-US">
      <Stack direction="column" gap="200">
        <Text fontWeight="600">US Section</Text>
        <DatePicker label="Date" />
      </Stack>
    </NimbusI18nProvider>

    <NimbusI18nProvider locale="de-DE">
      <Stack direction="column" gap="200">
        <Text fontWeight="600">German Section</Text>
        <DatePicker label="Datum" />
      </Stack>
    </NimbusI18nProvider>

    <NimbusI18nProvider locale="ja-JP">
      <Stack direction="column" gap="200">
        <Text fontWeight="600">Japanese Section</Text>
        <DatePicker label="日付" />
      </Stack>
    </NimbusI18nProvider>
  </Stack>
);
```
