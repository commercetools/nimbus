# Feedback

Components for providing user feedback.

- **Notifications**: e.g., Toast, Alert.
- **Status Indicators**: e.g., Spinner, ProgressBar.
- **Modals & Overlays**: e.g., Modal, Tooltip, Popover.
- **Validation**: e.g., ErrorMessage, SuccessMessage.
