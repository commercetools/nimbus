# Navigation

Components for moving through an application.

- **Menus**: e.g., Dropdown, ContextMenu.
- **Navigation Bars**: e.g., Navbar, Sidebar, Breadcrumbs.
- **Tabs & Accordions**: e.g., TabPanel, Accordion.
- **Pagination**: e.g., Paginator, InfiniteScroller.
