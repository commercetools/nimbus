# Typography

Below you'll find the raw tokens to change typography styles on a text-element.

## Text Styles

Text styles are collections of multiple typography rules (such as `font-size`, `line-height`, `letter-spacing` etc.) that create a specific, reusable style.

> [!NOTE]  
> I added `caption`, `detail` & `body` from our existing styles, but they are essentially just combinations
> of font-size & line-height and can be mapped to `xs`, `sm` & `md`. The `headline` looks can be achieved by using the
> textStyles on the `<Heading/>` component.

```jsx
<Text textStyle="nameOfDesiredTextStyle">Heading 1</Text>
```
