# Tooltip

A contextual popup that displays a description for an element.

[React Aria Components Tooltip Documentation](https://react-spectrum.adobe.com/react-aria/Tooltip.html)
[Tooltip Aria Pattern](https://www.w3.org/TR/wai-aria-1.2/#tooltip)

## Basic Usage

All instances of `Tooltip` consist of two components, the `TooltipTrigger` and
the `Tooltip`.

The `TooltipTrigger` is required to wrap both the trigger element and the
`Tooltip`.

```jsx-live
const App = () => (
  <TooltipTrigger>
    <Button>Trigger Element</Button>
    <Tooltip>Tooltip</Tooltip>
  </TooltipTrigger>
  )
```

## Use With Custom Trigger Element

All trigger elements must use the `useButton` or `useFocusable` hook from
`react-aria`, if you would like to use the `Tooltip` with a custom component,
you can wrap the custom element in the `FocusableTooltipTrigger` element.

```jsx-live
const App = () => (
  <TooltipTrigger>
    <FocusableTooltipTrigger>
        <button>Custom Trigger Element</button>
    </FocusableTooltipTrigger>
    <Tooltip>Tooltip</Tooltip>
  </TooltipTrigger>
  )
```
