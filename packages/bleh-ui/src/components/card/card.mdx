### Sizes

Available sizes.

```jsx-live
const App = () => {

  const sizes = ['2xl', 'xl', 'lg', 'md', 'sm', 'xs', '2xs'];

  return (
    <Stack direction="horizontal" alignItems="center">
      {sizes.map(size => (
        <Card key={size} size={size}>'{size}' Card</Card>

    </Stack>
```

Available variants.

```jsx-live
const App = () => {
```

```jsx-live
const App = () => {
```
