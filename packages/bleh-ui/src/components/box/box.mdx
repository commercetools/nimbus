# Box

Box is a basic layout component that serves as a wrapper or container. It's based on the Chakra UI Box component and supports all style props for rapid UI development.

## Import

```jsx
import { Box } from '@bleh-ui/react';
```

## Usage

### Basic Usage

```jsx
<Box p="4" bg="primary.3">
  Basic Box
</Box>
```

### As Flex Container

```jsx
<Box display="flex" gap="4">
  <Box p="2" bg="primary.3">Item 1</Box>
  <Box p="2" bg="primary.3">Item 2</Box>
</Box>
```

### With Border

```jsx
<Box p="4" border="1px" borderColor="primary.7">
  Box with border
</Box>
```

### With Shadow

```jsx
<Box p="4" boxShadow="md" bg="primary.3">
  Box with shadow
</Box>
```

### With asChild

Use the `asChild` prop to apply Box styles to any child element while maintaining the child's functionality:

```jsx
<Box p="4" bg="primary.3" asChild>
  <button onClick={() => alert("Clicked!")}>
    I am a button with Box styles
  </button>
</Box>
```

## Props

The Box component accepts all system props from Chakra UI, including:

- Space props (margin, padding)
- Color props (background, color)
- Typography props (fontSize, fontWeight)
- Layout props (width, height, display)
- Flexbox props
- Grid props
- Border props
- Position props
- Shadow props

## Accessibility

Box renders a `div` by default but can be changed to any other HTML element using the `as` prop.

```jsx
<Box as="section" p="4">
  Section content
</Box>
```
