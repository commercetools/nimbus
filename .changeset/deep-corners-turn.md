Adds new @commercetools/nimbus-design-token-ts-plugin package to enable
displaying the css value of design tokens in ts intellisense autocomplete. For
example, when you type `gap="`, the dropdown shows `600 = 24px` instead of just
`600`. Also adds documentation for how to install and use this package in the
typescript support section of the nimbus installation guide -
https://nimbus-documentation.vercel.app/home/getting-started/installation#typescript-support
