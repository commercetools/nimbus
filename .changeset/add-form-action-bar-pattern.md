Add FormActionBar pattern — a flat-props wrapper that composes save, cancel, and
optional delete buttons for form footers. Works inside any footer slot
(`DefaultPage.Footer`, `ModalPage.Footer`, `Drawer.Footer`, `Dialog.Footer`).

Non-English locales currently ship with English placeholder strings; final
translations will land via Transifex.
