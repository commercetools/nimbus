`Splitter`: new compound component for user-resizable two-pane layouts. A
draggable, keyboard-operable handle sits between two panes, which can be
collapsible and laid out horizontally or vertically. Sizes are uncontrolled by
default (`defaultSizes`) or controllable in place via the `sizes` prop for
responsive, per-breakpoint layouts. Layout persistence is wired to your own
storage. Nest splitters for three or more regions. See the docs for the full
API.
